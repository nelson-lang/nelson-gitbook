

# ismacro

Check for the existence of a macro (function).

## Syntax

- tf = ismacro(name)

## Input argument

 - name - a string: macro name.

## Output argument

 - tf - a logical: true if macro exists.

## Description


  <p><b>isbuiltin</b> checks for the existence of a macro.</p>


## Example

```Nelson
ismacro('isbuiltin')
ismacro('exist')
```
