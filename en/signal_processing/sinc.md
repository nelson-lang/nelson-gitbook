# sinc

Sinc function.

## Syntax

- c = sinc(m)

## Input argument

- m - input array: scalar, vector or matrix.

## Output argument

- c - sinc of input

## Description

  <p><b>c = sinc(m)</b> returns an array <b>c</b> whose elements are the sinc of the elements of the input: <b>m</b>.</p>

## Example

```matlab
c = sinc(pi)
```
