# bartlett

Bartlett window.

## Syntax

- c = bartlett(m)

## Input argument

- m - positive integer: window length

## Output argument

- c - column vector

## Description

  <p><b>c = bartlett(m)</b> an L-point symmetric Bartlett window.</p>

## Example

```matlab
c = bartlett(8)
```
