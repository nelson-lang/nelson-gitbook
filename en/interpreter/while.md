# while

while loop.

## Syntax

- while test_expression, statements, end

## Description

  <p><b>while</b> loop executes a set of statements as long as a the test condition remains <b>true</b>.</p>

## Example

```matlab
i = 0;
while lt(i, 10)
  disp(i)
  i = i + 1;
end
```
