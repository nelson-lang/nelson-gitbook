# keyboard

Stops script execution and enter in debug mode.

## Syntax

- keyboard()

## Description

  <p><b>keyboard</b> stops script execution and enter in debug mode. prompt is modified and displays debug level.</p>

## Example

```matlab
keyboard()
```
