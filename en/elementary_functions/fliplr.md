

# fliplr

Flip order of elements left to right

## Syntax

- B = fliplr(A)

## Input argument

 - A - an array

## Output argument

 - B - flipped array.

## Description


  <p><b>fliplr</b> return an new array of <b>A</b> flipped left to right.</p>


## Example

```matlab
x = eye(3, 2);
y = fliplr(x)
```
