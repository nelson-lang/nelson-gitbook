

# mtimes

Element wise multiplication, .* operator

## Syntax

- C = times(A, B)
- C = A .* B

## Input argument

 - A - a variable
 - B - a variable

## Output argument

 - C - result of A .* B

## Description


  <p><b>C = times(A, B)</b> performs element wise multiplication operation: A .* B.</p>


## Examples

```Nelson
times(3, 4)
3 .* 4
```
```Nelson
M1 = [2 6 10; 4 8 70];
M2 = [-25 88 1; 23 29 41];
M1 .* M2
```
