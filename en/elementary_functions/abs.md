

# abs

Absolute value

## Syntax

- R = abs(M)

## Input argument

 - M - a variable

## Output argument

 - R - result of abs: absolute value.

## Description


  <p><b>abs</b> computes the absolute value.</p>
  <p>If input argument is a complex number, <b>abs</b> computes the complex magnitude.</p>


## Example

```matlab
x = [1+i,-i;i,2i];
r = abs(x)
```
