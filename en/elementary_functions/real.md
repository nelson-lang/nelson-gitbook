# real

Real part of an complex number.

## Syntax

- R = real(M)

## Input argument

- M - a variable

## Output argument

- R - real part of the elements of the complex array M.

## Description

  <p><b>R = real(M)</b> Return the real part of M.</p>

## Example

```matlab
cplx = 22+34*i;
r = real(cplx)
```
