# factorial

Factorial function

## Syntax

- R = factorial(M)

## Input argument

- M - a integer, real single or real double matrix.

## Output argument

- R - result of factorial function.

## Description

  <p><b>factorial</b> computes the factorial function: the product of all integers values: 1 * 2 * ... * M</p>

## Example

```matlab
R = factorial([1:10])
R = factorial(int8(4))
```
