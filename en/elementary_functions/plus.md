

# plus

Addition, + operator

## Syntax

- C = plus(A, B)
- C = A + B

## Input argument

 - A - a variable
 - B - a variable

## Output argument

 - C - result of A + B

## Description


  <p><b>C = plus(A, B)</b> performs addition A + B variables.</p>


## Examples

```Nelson
plus(3, 4)
3 + 4
```
```Nelson
[1, 2] + 1
plus([1, 2], 1)
```
```Nelson
ones(0, 0) + 1
```
