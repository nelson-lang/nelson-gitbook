

# uplus

Unary plus, + operator

## Syntax

- C = uplus(A)
- C = +A

## Input argument

 - A - a variable

## Output argument

 - C - result of +A

## Description


  <p><b>C = uplus(A)</b> performs unary plus ie +A.</p>


## Example

```Nelson
M =-3;
+M
```
