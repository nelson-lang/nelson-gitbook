# cov

Covariance

## Syntax

- C = cov(M)

## Input argument

- M - a vector or matrix

## Output argument

- V - Covariance of M.

## Description

  <p><b>C = cov(M)</b> returns the covariance.</p>

## Example

```matlab
M = [4 -7 3; 1 4 -2; 10 7 9];
C = cov(M)
```
