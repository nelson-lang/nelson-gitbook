# acotd

Inverse cotangent of angle in degrees

## Syntax

- res = acotd(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

<b>acotd</b> computes the inverse cotangent of angle for each element of <b>x</b> in degrees.

## Example

```matlab
R = acotd([-i pi+i*pi/2 -1+i*4])
```
