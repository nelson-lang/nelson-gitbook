# asinh

Inverse hyperbolic sine function

## Syntax

- res = asinh(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

<b>cosh</b> computes the inverse hyperbolic sine in radians for each element of <b>x</b>.

## Example

```matlab
A = eye(3, 3);
res = asinh(A)
```
