# sinm

Computes the matrix sinus of a square matrix.

## Syntax

- res = sinm(x)

## Input argument

- x - a numeric value: scalar or square matrix

## Output argument

- res - a numeric value: a square matrix

## Description

<b>sinm(x)</b> computes the matrix sinus of <b>x</b>.

## Example

```matlab
A = eye(3, 3);
res = sinm(A)
A = [1, 2; 3, 4];
res = sinm(A)
```
