# acot

Inverse cotangent of angle in radians

## Syntax

- res = acot(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

<b>acot</b> computes the inverse cotangent of angle for each element of <b>x</b>.

## Example

```matlab
R = acot([-i pi+i*pi/2 -1+i*4])
```
