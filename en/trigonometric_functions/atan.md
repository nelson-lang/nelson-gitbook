

# atan

Computes the inverse tangent in radians for each element of x.

## Syntax

- res = atan(x)

## Input argument

 - x - a numeric value

## Output argument

 - res - a numeric value

## Description

<b>atan</b> computes the inverse tangent in radians for each element of x.

## Example

```Nelson
A = eye(3, 3);
res = atan(A)
```
