

# sinh

Computes the hyperbolic sine in radians for each element of x.

## Syntax

- res = sinh(x)

## Input argument

 - x - a numeric value

## Output argument

 - res - a numeric value

## Description

<b>sinh</b> computes the hyperbolic sine in radians for each element of x.

## Example

```Nelson
A = eye(3, 3);
res = sinh(A)
```
