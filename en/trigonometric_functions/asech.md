# asech

Inverse hyperbolic secant of angle in radians.

## Syntax

- res = asech(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

<b>asech</b> computes the inverse hyperbolic secant of argument in radians for each element of <b>x</b>.

## Example

```matlab
x = -pi:0.75:pi;
R = asech(x)
```
