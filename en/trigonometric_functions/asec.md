# asec

Inverse secant of angle in radians.

## Syntax

- res = asec(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

<b>asec</b> computes the inverse secant of argument in radians for each element of <b>x</b>.

## Example

```matlab
x = -pi:0.75:pi;
R = asec(x)
```
