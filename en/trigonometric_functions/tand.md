# tand

Computes the tangent in degree for each element of x.

## Syntax

- res = tand(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

<b>tand</b> computes the tangent in degree for each element of x.

## Example

```matlab
A = [0 30 45 60 90 360];
res = tand(A)
```
