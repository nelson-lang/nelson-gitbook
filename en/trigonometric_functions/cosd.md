# cosd

Computes the cosine in degree for each element of x.

## Syntax

- res = cosd(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

<b>cosd</b> computes the cosine in degree for each element of x.

## Example

```matlab
A = [0 30 45 60 90 360];;
res = cosd(A)
```
