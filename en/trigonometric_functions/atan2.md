# atan2

Computes the four-quadrant inverse tangent.

## Syntax

- res = atan2(y, x)

## Input argument

- y - a numeric value (double or single real)
- x - a numeric value (double or single real)

## Output argument

- res - a numeric value

## Description

<b>atan2</b> computes the four-quadrant inverse tangent.

## Example

```matlab
atan2(1, 0)
```
