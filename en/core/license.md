# license

Get license information for Nelson.

## Syntax

- license
- r = license
- [r, txt] = license

## Output argument

- r - a string: minimal string description about license
- txt - a string: complete license text.

## Description

  <p><b>license</b> get license information for Nelson.</p>

## Example

```matlab
license()
r = license()
[r,txt] = license()
```
