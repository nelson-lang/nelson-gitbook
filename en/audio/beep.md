# beep

Produces a beep sound.

## Syntax

- beep
- beep(str)
- str = beep

## Input argument

- str - a string: 'on' or 'off'.

## Output argument

- str - a string: 'on' or 'off'.

## Description

<b>beep</b> produces an beep system.

## Example

```matlab
beep('off')
beep
beep('on')
beep
s = beep
```
