# getlanguage

Returns the current language in Nelson.

## Syntax

- lang = getlanguage()

## Output argument

- lang - a string: current language used in Nelson.

## Description

  <p><b>getlanguage</b> returns the current language used in Nelson.</p>

## Example

```matlab
l = getlanguage()
```
