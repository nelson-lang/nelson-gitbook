# MPI_Get_version

Return the version number of MPI.

## Syntax

- [major, minor] = MPI_Get_version()

## Output argument

- major - an integer value.
- minor - an integer value.

## Description

  <p>Return the version number of MPI.</p>

## See also

[MPI_Init](MPI_Init.md), [MPI_Finalize](MPI_Finalize.md).

## Example

```matlab
if ~MPI_Initialized()
  MPI_Init();
end
[major, minor] = MPI_Get_version()
if MPI_Initialized()
  MPI_Finalize();
end
```
