# qml_collectgarbage

Runs the Qml garbage collector.

## Syntax

- qml_collectgarbage

## Description

  <p>The garbage collector will attempt to reclaim memory by locating and disposing of objects that are no longer reachable in the script environment.</p>

## Example

```matlab
qml_collectgarbage()
```
