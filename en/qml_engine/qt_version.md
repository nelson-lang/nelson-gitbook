# qt_version

Returns Qt version used.

## Syntax

- v = qt_version()

## Output argument

- v - a string : valid path.

## Description

  <p><b>v = qt_version()</b> returns the version number of Qt at run-time as a string (for example, "5.15.2").</p>

## Example

```matlab
semver(qt_version(), '>5.15')
```
