# transpose

Returns vector or matrix transpose: .' operator.

## Syntax

- C= transpose(A)
- C = A .'

## Input argument

- A - a variable

## Output argument

- C - result: transpose of A.

## Description

  <p><b>C = transpose(A)</b> returns the transpose of A.</p>

## Examples

```matlab
A = 3
B = A.'
```

```matlab
A = -i
B = A.'
```

```matlab
A = sparse(eye(3, 4) * i)
B = A.'
```
