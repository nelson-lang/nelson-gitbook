

# ne

Inequality, ~= operator

## Syntax

- C = ne(A, B)
- C = A ~= B

## Input argument

 - A - a variable
 - B - a variable

## Output argument

 - C - result of A ~= B

## Description


  <p><b>C = ne(A, B)</b> performs inequality operation: A ~= B variables.</p>


## Example

```matlab
ne(3, 4)
3 ~= 4
```
