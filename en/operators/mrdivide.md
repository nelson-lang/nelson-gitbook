# mrdivide

Matrix right division, / operator.

## Syntax

- C = mrdivide(A, B)
- C = A / B

## Input argument

- A - a variable
- B - a variable

## Output argument

- C - result of A / B

## Description

  <p><b>C = mrdivide(A, B)</b> returns the matrix right division of A and B.</p>

## Example

```matlab
B = ones(3, 4)
A = B *2
A / B
```
