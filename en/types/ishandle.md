# ishandle

Return true if variable var is a handle object.

## Syntax

- res = ishandle(var)

## Input argument

- var - a variable

## Output argument

- res - a logical: true or false

## Description

<b>ishandle</b> returns a logical 1 if the argument is a handle object and a logical 0 otherwise.

## Example

```matlab
A = 3;
res = ishandle(A)
```
