# islogical

Return true if variable var is a logical.

## Syntax

- res = islogical(var)

## Input argument

- var - a variable

## Output argument

- res - a logical: true or false

## Description

<b>islogical</b> returns a logical 1 if the argument is a logical array and a logical 0 otherwise.

## Examples

```matlab
A = 1;
res = islogical(A)
```

```matlab
B = logical(1);
res = islogical(B)
```
