# getdynlibext

Returns the extension of dynamic libraries.

## Syntax

- ext = getdynlibext()

## Output argument

- ext - a string: dynamic library extension

## Description

  <p><b>getdynlibext()</b> returns the extension of dynamic libraries.</p>

## Example

```matlab
getdynlibext()
```
