

# Supported C/C++ compilers

## Description


  <p>
    <b>Windows:</b>
  </p>
  <p>- Microsoft Visual Studio C/C++ 2017 all versions (Pro, Enterprise or Community)</p>
  <p>- MinGW</p>
  <p>
    <b>GNU/Linux:</b>
  </p>
  <p>- GNU C/C++ compilers packaged with all distributions.</p>
  <p>
    <b>MacOS X:</b>
  </p>
  <p>- XCode C/C++ compilers.</p>
