# flintmax

Largest consecutive integer in floating-point format.

## Syntax

- R = flintmax()
- R = flintmax('double')
- R = flintmax('single')
- R = flintmax('like', V)

## Input argument

- V - a double or single variable.

## Output argument

- R - a double or single.

## Description

  <p><b>flintmax</b> returns largest consecutive integer in floating-point format.</p>

## Example

```matlab
flintmax
flintmax('double')
flintmax('like', pi)
flintmax('single')
```
