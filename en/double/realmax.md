# realmax

Largest positive floating-point number.

## Syntax

- R = realmax()
- R = realtmax('double')
- R = realmax('single')

## Output argument

- R - a double or single.

## Description

  <p><b>realmax</b> returns largest positive floating-point number.</p>

## Example

```matlab
realmax
realmax('double')
realmax('single')
```
