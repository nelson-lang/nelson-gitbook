# realmin

Smallest positive floating-point number.

## Syntax

- R = realmin()
- R = realmin('double')
- R = realmin('single')

## Output argument

- R - a double or single.

## Description

  <p><b>realmin</b> returns smallest positive floating-point number.</p>

## Example

```matlab
realmin
realmin('double')
realmin('single')
```
