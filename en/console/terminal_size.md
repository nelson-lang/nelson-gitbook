

# terminal_size

Query the size of the terminal window.

## Syntax

- [r, c] = terminal_size()

## Output argument

 - [r, c] - a vector: rows and columns

## Description


  <p><b>terminal_size()</b> returns a vector with size of the terminal window in characters (rows and columns).</p>


## Example

```matlab
terminal_size()
```
