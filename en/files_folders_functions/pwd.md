

# pwd

Returns current directory.

## Syntax

- pwd()
- r = pwd()

## Output argument

 - r - a string: current directory.

## Description


  <p>Returns the current working directory.</p>
  <p><b>pwd()</b> without input argument displays the current working directory.</p>
  <p/>


## Example

```Nelson
r = pwd()
pwd()
```
