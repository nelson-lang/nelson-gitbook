

# single

Converts a variable to single precision type.

## Syntax

- S = single(V)

## Input argument

 - V - a variable.

## Output argument

 - S - a single.

## Description


  <p><b>single(V)</b> converts to the single-precision type.</p>


## Examples

```Nelson
single('Nelson')
```
```Nelson
A = single(pi)
```
