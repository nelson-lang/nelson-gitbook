# isspace

Determine which characters are space.

## Syntax

- res = isspace(str)

## Input argument

- str - scalar, vector, matrix or multidimensional array.

## Output argument

- res - logical array

## Description

<b>isletter</b> determines which characters are space characters.

## Examples

```matlab
isspace('Nel Son')
```

```matlab
isspace("六書 six writings")
```
