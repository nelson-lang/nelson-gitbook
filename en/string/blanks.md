

# blanks

creates an string of blank characters.

## Syntax

- r = blanks(n)

## Input argument

 - n - an value integer, number of blanks.

## Output argument

 - r - a character array with n blanks

## Description

<b>blanks</b> creates an string of blank characters.

## Example

```matlab
blanks(4)
```
