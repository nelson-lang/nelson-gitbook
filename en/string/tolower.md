# tolower

Lower case conversion.

## Syntax

- res = tolower(str)

## Input argument

- str - a row character array, a cell of strings or an string array.

## Output argument

- res - lower case equivalent

## Description

<b>tolower</b> converts a string to lower case.

## Examples

```matlab
tolower('NelSon')
```

```matlab
tolower(["NelSon", "is", "open"])
```
