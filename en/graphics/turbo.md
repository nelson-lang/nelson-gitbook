# turbo

Turbo colormap array.

## Syntax

- c = turbo
- c = turbo(m)

## Input argument

- m - a scalar integer value: Number of colors (256 as default value).

## Output argument

- c - Turbo colormap array.

## Description

  <p><b>turbo</b> returns the colormap with turbo colors.</p>

## Example

```matlab
f = figure();
surf(peaks);
colormap('turbo');
```
