# sky

Sky colormap array.

## Syntax

- c = sky
- c = sky(m)

## Input argument

- m - a scalar integer value: Number of colors (256 as default value).

## Output argument

- c - Sky colormap array.

## Description

  <p><b>sky</b> returns the colormap with sky colors.</p>

## Example

```matlab
f = figure();
surf(peaks);
colormap('sky');
```
