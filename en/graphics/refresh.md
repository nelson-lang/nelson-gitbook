# refresh

Redraw current figure.

## Syntax

- refresh()
- refresh(F)

## Input argument

- F - figure graphics object.

## Description

  <p><b>refresh</b> erases and redraws the current figure.</p>
  <p><b>refresh(F)</b> redraws the figure identified by <b>F</b>.</p>
