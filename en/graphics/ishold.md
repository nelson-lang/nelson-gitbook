# ishold

Get current hold state.

## Syntax

- tf = ishold()
- tf = ishold(ax)

## Input argument

- ax - scalar graphics object: axes.

## Output argument

- tf - a scalar logical: true if it is hold on.

## Description

  <p><b>tf = ishold(ax)</b> returns the hold state of the specified axes object.</p>
