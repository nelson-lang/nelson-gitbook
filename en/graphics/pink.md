# pink

Pink colormap array.

## Syntax

- c = pink
- c = pink(m)

## Input argument

- m - a scalar integer value: Number of colors (256 as default value).

## Output argument

- c - Pink colormap array.

## Description

  <p><b>pink</b> returns the colormap with pink colors.</p>

## Example

```matlab
f = figure();
surf(peaks);
colormap('pink');
```
