

# disp

Display a variable.

## Syntax

- disp(V)

## Input argument

 - V - a variable

## Description


  <p><b>disp(V)</b> displays the value of the variable <b>V</b>.</p>
  <p><b>disp</b> uses current <b>format</b> setting to display numeric values.</p>


## Examples

```matlab
disp('Hello Nelson')
```
```matlab
disp(pi)
```
```matlab
disp(eye(3, 3))
```
disp always ends with a newline.
```matlab
disp('')
```
