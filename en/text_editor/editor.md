# editor

call the embedded text editor.

## Syntax

- editor()
- editor(filename)

## Input argument

- filename - a string: filename to open.

## Description

  <p><b>editor</b> opens an existing file in the nelson's editor.</p>
  <p><b>editor</b> must be considered as internal.</p>
