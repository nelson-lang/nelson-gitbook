# isquietmode

Return true if Nelson started with --quiet option.

## Syntax

- res = isquietmode()

## Output argument

- res - a logical true or false

## Description

  <p><b>isquietmode</b> returns a logical 1 if Nelson started with --quiet option and a logical 0 otherwise.</p>

## Example

```matlab
disp(isquietmode());
```
