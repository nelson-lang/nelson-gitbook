# qml_createqquickview

Charge un fichier QML et crée une fenêtre.

## Syntaxe

- h = qml_createqquickview(filename)

## Argument d'entrée

- filename - une chaîne : nom de fichier QML.

## Argument de sortie

- h - une poignée (handle) QObject.

## Description

<p>Charge un fichier QML</p>

<p>Il crée un composant QML, une fenêtre et charge le fichier .qml.</p>
