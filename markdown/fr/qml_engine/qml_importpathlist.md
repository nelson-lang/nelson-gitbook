# qml_importpathlist

Renvoie la liste des répertoires où le moteur recherche les modules installés dans une structure de répertoires basée sur des URL.

## Syntaxe

- p = qml_importpathlist()

## Argument de sortie

- p - un tableau de chaînes : chemins.

## Description

<p>Renvoie la liste des répertoires où le moteur recherche les modules installés dans une structure de répertoires basée sur des URL.</p>

## Exemple

```matlab
qml_importpathlist()
```
