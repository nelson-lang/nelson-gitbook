# qml_clearcomponentcache

Vide le cache interne de composants du moteur.

## Syntaxe

- qml_clearcomponentcache

## Description

<p>Cette fonction provoque la destruction des métadonnées de propriété de tous les composants précédemment chargés par le moteur.</p>

<p>Tous les composants précédemment chargés et les liaisons de propriété pour tous les objets existants créés à partir de ces composants cesseront de fonctionner.</p>

## Exemple

```matlab
qml_clearcomponentcache()
```
