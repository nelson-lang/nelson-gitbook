# QObject_root

Objet racine QObject.

## Syntaxe

- r = QObject_root()

## Argument de sortie

- h - poignée (handle) QObject de l'interface graphique Nelson.

## Description

<p>Renvoie la poignée (handle) QObject de l'interface graphique Nelson.</p>

## Exemple

```matlab
h1 = QObject_root()
h1.windowTitle
h1.windowTitle = 'Your title'
```
