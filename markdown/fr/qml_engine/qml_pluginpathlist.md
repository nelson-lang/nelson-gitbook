# qml_pluginpathlist

Renvoie la liste des répertoires où le moteur recherche les plugins natifs pour les modules importés.

## Syntaxe

- p = qml_pluginpathlist()

## Argument de sortie

- p - un tableau de chaînes : chemins.

## Description

<p>Renvoie la liste des répertoires où le moteur recherche les plugins natifs pour les modules importés.</p>

## Exemple

```matlab
qml_pluginpathlist()
```
