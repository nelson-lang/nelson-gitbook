# nelsonObject

objet nelson appelable depuis QML.

## Syntaxe

- nelson.disp(msg)
- nelson.evaluate(cmd)
- nelson.processevent()
- nelson.call(function_name)
- nelson.call(function_name, arg1, ..., arg5)

## Argument d'entrée

- msg - une chaîne.
- cmd - une chaîne.
- function_name - une chaîne : nom de la fonction nelson à appeler
- arg1, ..., arg5 - variables JavaScript

## Description

<p>
                        nelson contient des méthodes utilisées comme callbacks pour appeler nelson depuis QML</p>
