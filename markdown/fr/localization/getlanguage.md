# getlanguage

Renvoie la langue courante dans Nelson.

## Syntaxe

- lang = getlanguage()

## Argument de sortie

- lang - une chaîne : langue courante utilisée dans Nelson.

## Description

<p>
            getlanguage renvoie la langue courante utilisée dans Nelson.</p>

## Exemple

```matlab
l = getlanguage()
```
