# getdefaultlanguage

Renvoie la langue par défaut utilisée dans Nelson.

## Syntaxe

- lang = getdefaultlanguage()

## Argument de sortie

- lang - une chaîne : 'fr_FR' par défaut.

## Description

<p>
            getdefaultlanguage renvoie la langue par défaut utilisée par Nelson.</p>

## Exemple

```matlab
getdefaultlanguage()
```
