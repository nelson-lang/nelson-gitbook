# setlanguage

Modifie la langue utilisée dans Nelson.

## Syntaxe

- setlanguage(language)

## Argument d'entrée

- language - une chaîne : 'fr_FR', 'fr_FR' ou d'autres par défaut.

## Description

<p>
            setlanguage modifie la langue utilisée par Nelson et enregistre ce changement pour les exécutions ultérieures de Nelson.</p>
