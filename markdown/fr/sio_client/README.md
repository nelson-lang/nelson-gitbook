# Nelson dans le cloud

Ce module fournit des fonctions pour gérer les connexions, envoyer et recevoir des données, et gérer les événements dans un environnement cloud.

## Functions

- [sioemit](sioemit.md) - Émettre un événement vers le client web.
- [siogetvariable](siogetvariable.md) - Émettre la valeur d'une variable vers le navigateur web.
