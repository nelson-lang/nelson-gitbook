# siogetvariable

Émettre la valeur d'une variable vers le navigateur web.

## Syntaxe

- siogetvariable(variable_name)

## Argument d'entrée

- variable_name - une chaîne : nom de la variable

## Description

<p>
            siogetvariable envoie la valeur d'une variable vers le navigateur web.</p>

## Exemple

```matlab
A = eye(3, 3); siogetvariable('A')
```
