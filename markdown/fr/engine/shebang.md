# #! shebang

Sur Unix/Linux, analyse la première ligne du script comme directive d'interpréteur.

## Description

<p>Sur Unix, Linux et MacOS, le shebang permet d'exécuter directement un script NelSon.</p>

## Exemple

```matlab
#!nelson-adv-cli -q -f
argv()
disp('shebang example line 1')
disp('shebang example line 2')
exit()

```
