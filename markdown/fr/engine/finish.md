# finish

Script de terminaison défini par l'utilisateur pour Nelson.

## Description

<p>
            startup.m dans Nelson permet d'initialiser des commandes spécifiées par l'utilisateur au démarrage de Nelson.</p>

<p>Il exécute tout fichier nommé startup.m qui se trouve dans le chemin de recherche.</p>

<p>Pour utiliser cette fonctionnalité, créez un fichier nommé startup.m dans le dossier userpath, qui fait partie du chemin de recherche de Nelson.</p>

<p>Insérez dans ce fichier les commandes que vous souhaitez voir exécutées au démarrage de Nelson.</p>

<p>Cela peut inclure la définition de constantes physiques, des valeurs par défaut pour les propriétés graphiques, l'ajout de facteurs de conversion, ou la pré-définition d'autres éléments souhaités dans votre espace de travail.</p>

<p>La personnalisation du fichier startup.m vous permet d'établir un environnement adapté à chaque lancement de Nelson.</p>
