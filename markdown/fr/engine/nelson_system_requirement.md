# Exigences système

Exigences système par plateforme.

## Description

<p>Linux :</p>

<p>Système d'exploitation : Ubuntu 24.04 LTS, Ubuntu 22.04 LTS, Ubuntu 20.04 LTS, Fedora, ArchLinux, NixOs</p>

<p>Processeur : tout processeur Intel ou AMD x86-64.</p>

<p>Mémoire (RAM) : voir les exigences du système d'exploitation (16 Go recommandés ou plus).</p>

<p>Stockage : 1 Go pour l'installation de tous les produits ; un SSD/NVMe est recommandé.</p>

<p>Graphisme : un GPU avec accélération matérielle et support OpenGL et au moins 1 Go de mémoire est recommandé.</p>

<p></p>

<p>Windows :</p>

<p>Système d'exploitation : Windows 10, 11 ou Windows Server 2022.</p>

<p>Processeur : tout processeur Intel ou AMD x86-64 avec jeu d'instructions AVX2 (CPU publié à partir de 2015).</p>

<p>Mémoire (RAM) : voir les exigences du système d'exploitation (16 Go recommandés ou plus).</p>

<p>Stockage : 1 Go pour l'installation de tous les produits ; un SSD/NVMe est recommandé.</p>

<p>Graphisme : un GPU avec accélération matérielle et support OpenGL et au moins 1 Go de mémoire est recommandé.</p>

<p></p>

<p>MacOs :</p>

<p>Système d'exploitation : macOS Tahoe, macOS Sequoia, macOS Sonoma, macOS Ventura.</p>

<p>Processeur : tout processeur Intel supporté par Apple et tout processeur de la série M.</p>

<p>Stockage : 1 Go pour l'installation de tous les produits ; un SSD/NVMe est recommandé.</p>

<p>Graphisme : tout Mac capable d'exécuter macOS Ventura dispose d'un GPU pouvant exécuter Nelson.</p>

<p></p>
