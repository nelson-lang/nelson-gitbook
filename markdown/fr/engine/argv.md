# argv

Arguments de la ligne de commande de Nelson.

## Syntaxe

- args = argv()

## Argument de sortie

- args - un tableau de cellules contenant des chaînes de caractères.

## Description

<p>
            argv() renvoie un tableau de cellules contenant les arguments de la ligne de commande de Nelson.</p>

<p>Le premier élément du tableau retourné contient le chemin de l'exécutable lancé.</p>

## Exemple

```matlab
argv()
```
