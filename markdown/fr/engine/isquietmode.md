# isquietmode

Renvoie vrai si Nelson a été démarré avec l'option --quiet.

## Syntaxe

- res = isquietmode()

## Argument de sortie

- res - un booléen true ou false

## Description

<p>
            isquietmode renvoie true si Nelson a été démarré avec l'option --quiet et false sinon.</p>

## Exemple

```matlab
disp(isquietmode());
```
