# pi

Rapport de la circonférence d'un cercle à son diamètre.

## Syntaxe

- pi

## Description

<p>
            pi retourne le nombre à virgule flottante le plus proche de la valeur de π.</p>

## Exemples

```matlab
cos(pi)
```

```matlab
sin(pi)
```

```matlab
4*atan(1) == pi
```
