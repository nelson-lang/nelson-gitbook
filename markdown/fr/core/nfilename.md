# nfilename

Nom du fichier courant exécuté.

## Syntaxe

- R = nfilename()
- R = nfilename('fullpath')
- R = nfilename('fullpathext')

## Argument de sortie

- R - une chaîne : le chemin du fichier fonctionnant actuellement

## Description

<p>Renvoie le nom du fichier de script actuellement exécuté ou évalué.</p>
