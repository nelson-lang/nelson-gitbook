# feature

Interroge les fonctionnalités disponibles.

## Syntaxe

- ret = feature(name)
- ret = feature(name, newValue)

## Argument d'entrée

- name - chaîne : nom de la fonctionnalité
- newValue - une variable (optionnelle)

## Argument de sortie

- ret - résultat : valeur renvoyée

## Description

<p>Retourne des informations sur les fonctionnalités ou options disponibles dans l'environnement Nelson.</p>
