# license

Affiche les informations de licence.

## Syntaxe

- license
- r = license
- [r, txt] = license

## Argument de sortie

- r - a string: minimal string description about license
- txt - a string: complete license text.

## Description

<p>Affiche ou retourne les informations de licence associées à l'installation de Nelson.</p>

## Exemple

```matlab
license()
r = license()
[r,txt] = license()
```
