# isunicodesupported

Indique si Unicode est supporté.

## Syntaxe

- tf = isunicodesupported()

## Argument de sortie

- tf - un logique : vrai ou faux.

## Description

<p>Retourne vrai si l'environnement et la plateforme prennent en charge Unicode pour les chaînes de caractères.</p>

## Exemple

```matlab
isunicodesupported()
```
