# nelsonroot

Répertoire racine de Nelson.

## Syntaxe

- nelson_path = nelsonroot

## Argument de sortie

- nelson_path - a string: the root folder of Nelson.

## Description

<p>Renvoie le répertoire racine où Nelson est installé ou configuré.</p>

## Exemple

```matlab
pwd
cd(nelsonroot)
pwd
```
