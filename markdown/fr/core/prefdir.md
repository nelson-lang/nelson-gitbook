# prefdir

Répertoire des préférences utilisateur.

## Syntaxe

- pref_path = prefdir

## Argument de sortie

- pref_path - a string: the preferences directory

## Description

<p>Retourne le répertoire où sont stockées les préférences spécifiques à l'utilisateur pour Nelson.</p>

## Exemple

an example

```matlab
cd(prefdir)
```
