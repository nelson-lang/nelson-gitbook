# toolboxdir

Renvoie le chemin d'un module.

## Syntaxe

- p = toolboxdir(module_short_name)

## Argument d'entrée

- module_short_name - chaîne : nom court du module.

## Argument de sortie

- p - chaîne : chemin du module.

## Description

<p>
            toolboxdir est une fonction d'aide qui renvoie le chemin racine d'un module.</p>

## Exemple

```matlab
toolboxdir('core')
```
