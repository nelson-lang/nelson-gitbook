# tolower

Conversion en minuscules.

## Syntaxe

- res = tolower(str)

## Argument d'entrée

- str - un tableau de caractères (ligne), une cellule de chaînes ou un tableau de chaînes.

## Argument de sortie

- res - équivalent en minuscules

## Description

        tolower convertit une chaîne en minuscules.

## Exemples

```matlab
tolower('NelSon')
```

```matlab
tolower(["NelSon", "is", "open"])
```
