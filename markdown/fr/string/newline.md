# newline

Renvoie le caractère de nouvelle ligne.

## Syntaxe

- ch = newline()

## Argument de sortie

- ch - un caractère : équivalent à char(10)

## Description

        newline renvoie un caractère de nouvelle ligne.

## Exemple

```matlab
double(newline)
```
