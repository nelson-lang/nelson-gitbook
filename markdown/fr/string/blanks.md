# blanks

crée une chaîne de caractères d'espaces.

## Syntaxe

- r = blanks(n)

## Argument d'entrée

- n - un entier: nombre d'espaces.

## Argument de sortie

- r - un tableau de caractères contenant n espaces

## Description

        blanks crée une chaîne composée d'espaces.

## Exemple

```matlab
blanks(4)
```
