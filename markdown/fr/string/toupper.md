# toupper

Conversion en majuscules.

## Syntaxe

- res = toupper(str)

## Argument d'entrée

- str - un tableau de caractères (ligne), une cellule de chaînes ou un tableau de chaînes.

## Argument de sortie

- res - une chaîne en majuscules

## Description

           toupper convertit une chaîne en majuscules.

## Exemples

```matlab
toupper('NelSon')
```

```matlab
upper(["NelSon", "is", "open"])
```
