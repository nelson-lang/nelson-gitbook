# isspace

Détermine quels caractères sont des espaces.

## Syntaxe

- res = isspace(str)

## Argument d'entrée

- str - scalaire, vecteur, matrice ou tableau multidimensionnel.

## Argument de sortie

- res - le tableau logique

## Description

        isletter détermine quels caractères sont des espaces.

## Exemples

```matlab
isspace('Nel Son')
```

```matlab
isspace("六書 six writings")
```
