# isletter

Détermine quels caractères sont des lettres.

## Syntaxe

- res = isletter(str)

## Argument d'entrée

- str - scalaire, vecteur, matrice ou tableau multidimensionnel.

## Argument de sortie

- res - tableau logique

## Description

        isletter détermine quels caractères sont des lettres.

## Exemples

```matlab
isletter('Nel Son')
```

```matlab
isletter("六書 six writings")
```
