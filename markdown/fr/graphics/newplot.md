# newplot

Préparer la création d'un nouveau graphique.

## Syntaxe

- go = newplot()
- go = newplot(ax)

## Argument d'entrée

- ax - Figure ou axes spécifiés plutôt que la figure et les axes courants.

## Argument de sortie

- go - Objet graphique : type axes.

## Description

<p>
            newplot prépare une figure et des axes pour les commandes graphiques.
        </p>

## Exemple

```matlab
h = newplot()
```
