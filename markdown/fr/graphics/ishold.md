# ishold

Obtient l'état actuel du mode hold.

## Syntaxe

- tf = ishold()
- tf = ishold(ax)

## Argument d'entrée

- ax - Objet graphique scalaire : axes.

## Argument de sortie

- tf - Un scalaire logique : vrai si le mode hold est activé.

## Description

<p>
            tf = ishold(ax) retourne l'état du mode hold de l'objet axes spécifié.
        </p>
