# bone

Palette de couleurs bone.

## 📝 Syntaxe

- c = bone
- c = bone(m)

## 📥 Argument d'entrée

- m - une valeur entière scalaire : nombre de couleurs (256 par défaut).

## 📤 Argument de sortie

- c - Palette de couleurs bone.

## 📄 Description

<b>bone</b> retourne la palette de couleurs bone.

## 💡 Exemple

```matlab
f = figure();
surf(peaks);
colormap('bone');
```
