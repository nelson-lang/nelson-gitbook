# pink

Palette de couleurs Pink.

## Syntaxe

- c = pink
- c = pink(m)

## Argument d'entrée

- m - Valeur entière scalaire : nombre de couleurs (256 par défaut).

## Argument de sortie

- c - Palette de couleurs Pink.

## Description

<p>
            pink retourne la palette de couleurs Pink.
        </p>

## Exemple

```matlab
f = figure();
surf(peaks);
colormap('pink');
```
