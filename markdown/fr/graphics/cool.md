# cool

Palette de couleurs cool.

## 📝 Syntaxe

- c = cool
- c = cool(m)

## 📥 Argument d'entrée

- m - Un entier scalaire : nombre de couleurs (256 par défaut).

## 📤 Argument de sortie

- c - Palette de couleurs cool.

## 📄 Description

<b>cool</b> retourne la palette de couleurs cool.

## 💡 Exemple

```matlab
f = figure();
surf(peaks);
colormap('cool');
```
