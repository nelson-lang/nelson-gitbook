# cool

Palette de couleurs cool.

## Syntaxe

- c = cool
- c = cool(m)

## Argument d'entrée

- m - Un entier scalaire : nombre de couleurs (256 par défaut).

## Argument de sortie

- c - Palette de couleurs cool.

## Description

<p>
            cool retourne la palette de couleurs cool.</p>

## Exemple

```matlab
f = figure();
surf(peaks);
colormap('cool');
```
