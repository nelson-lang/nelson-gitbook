# abyss

Palette de couleurs abyss.

## 📝 Syntaxe

- c = abyss
- c = abyss(m)

## 📥 Argument d'entrée

- m - une valeur entière scalaire : nombre de couleurs (256 par défaut).

## 📤 Argument de sortie

- c - Palette de couleurs abyss.

## 📄 Description

<b>abyss</b> retourne la palette de couleurs abyss.

## 💡 Exemple

```matlab
f = figure();
surf(peaks);
colormap('abyss');
```
