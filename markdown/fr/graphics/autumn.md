# autumn

Palette de couleurs autumn.

## Syntaxe

- c = autumn
- c = autumn(m)

## Argument d'entrée

- m - une valeur entière scalaire : nombre de couleurs (256 par défaut).

## Argument de sortie

- c - Palette de couleurs autumn.

## Description

<p>
            autumn retourne la palette de couleurs autumn.</p>

## Exemple

```matlab
f = figure();
surf(peaks);
colormap('autumn');
```
