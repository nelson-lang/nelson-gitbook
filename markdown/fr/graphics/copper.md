# copper

Palette de couleurs copper.

## Syntaxe

- c = copper
- c = copper(m)

## Argument d'entrée

- m - Un entier scalaire : nombre de couleurs (256 par défaut).

## Argument de sortie

- c - Palette de couleurs copper.

## Description

<p>
            copper retourne la palette de couleurs copper.</p>

## Exemple

```matlab
f = figure();
surf(peaks);
colormap('copper');
```
