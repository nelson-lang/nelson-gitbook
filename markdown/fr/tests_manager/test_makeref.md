# test_makeref

Crée un fichier '.ref' pour un test

## Syntaxe

- status = test_makeref(filename)

## Argument d'entrée

- filename - a string: nom de fichier, un fichier de test.

## Argument de sortie

- status - un logique: vrai si le .ref a été généré.

## Description

<p>test_makeref crée un fichier '.ref' à partir d'un fichier de test.</p>

<p>Le fichier de test doit contenir la balise <--CHECK REF-->.</p>
