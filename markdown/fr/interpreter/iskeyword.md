# iskeyword

Renvoie tous les mots-clés de Nelson.

## Syntaxe

- state = iskeyword(name)
- ce = iskeyword()

## Argument d'entrée

- name - une chaîne.

## Argument de sortie

- state - un logique : true si c'est un mot-clé Nelson.
- ce - une cellule de chaînes : liste des mots-clés de Nelson.

## Description

<p>
            iskeyword renvoie la liste de tous les mots-clés de Nelson.</p>

## Exemple

```matlab
iskeyword('for')
ce = iskeyword()
```
