# keyboard

Arrête l'exécution du script et entre en mode débogage.

## Syntaxe

- keyboard()

## Description

<p>
            keyboard arrête l'exécution du script et entre en mode débogage. L'invite est modifiée et affiche le niveau de débogage.</p>

## Exemple

```matlab
 keyboard()
```
