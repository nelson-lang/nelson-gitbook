# minus

Subtraction, - operator

## Syntaxe

- C = minus(A, B)
- C = A - B

## Argument d'entr√©e

- A - a variable
- B - a variable

## Argument de sortie

- C - result of A - B

## Description

<p>
            C = minus(A, B) performs subtraction A - B variables.</p>

## Exemples

```matlab
minus(3, 4)
3 - 4
```

```matlab
[1, 2] - 1
minus([1, 2], 1)
```

```matlab
ones(0, 0) - 1
```
