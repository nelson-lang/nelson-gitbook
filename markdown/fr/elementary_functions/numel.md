# numel

Number of elements.

## Syntaxe

- nbel = numel(M)

## Argument d'entr√©e

- M - a variable

## Argument de sortie

- nbel - the number of elements.

## Description

<p>Return the number of elements in the object M.</p>

## Exemple

```matlab
numel(ones(3, 0))
numel(ones(3,4))
numel(ones(3,4,5))
```
