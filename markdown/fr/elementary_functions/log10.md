# log10

Common logarithm (base 10).

## Syntaxe

- R = log10(M)

## Argument d'entr√©e

- M - a variable

## Argument de sortie

- R - result of log: base 10.

## Description

<p>
            log10 computes common logarithm (base 10).</p>

<p>For negative real and complex values of M, log10 function returns complex values.</p>

## Exemple

```matlab
x = [1+i,-i;i,2i];
r = log10(x)
```
