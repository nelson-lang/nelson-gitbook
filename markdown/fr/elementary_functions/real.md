# real

Real part of an complex number.

## Syntaxe

- R = real(M)

## Argument d'entr√©e

- M - a variable

## Argument de sortie

- R - real part of the elements of the complex array M.

## Description

<p>
            R = real(M) Return the real part of M.</p>

## Exemple

```matlab
cplx = 22+34*i;
r = real(cplx)
```
