# floor

Arrondir vers le bas

## Syntaxe

- C = floor(A)

## Argument d'entrée

- A - une variable

## Argument de sortie

- C - résultat de floor.

## Description

<p>
            floor renvoie une matrice d'entiers obtenue en arrondissant chaque élément vers le bas.</p>

## Exemple

```matlab
floor(pi)
```
