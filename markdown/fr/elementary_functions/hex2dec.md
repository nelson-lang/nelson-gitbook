# hex2dec

Convertit un nombre en base 16 en décimal.

## Syntaxe

- D = hex2dec(TXT)

## Argument d'entrée

- TXT - un tableau de caractères.

## Argument de sortie

- D - résultat de hex2dec : une valeur entière.

## Description

<p>
            hex2dec convertit un nombre en base 16 en décimal.</p>

<p>Remarque : hex2dec et dec2hex sont des opérations réciproques.</p>

## Exemple

```matlab
bin2dec('11')
```
