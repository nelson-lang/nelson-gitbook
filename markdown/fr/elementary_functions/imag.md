# imag

Partie imaginaire d'un nombre complexe.

## Syntaxe

- im = imag(M)

## Argument d'entrée

- M - une variable

## Argument de sortie

- R - partie imaginaire des éléments du tableau complexe M.

## Description

<p>
            R = imag(M) Renvoie la partie imaginaire de M.</p>

## Exemple

```matlab
cplx = 22+34*i;
r = imag(cplx)
```
