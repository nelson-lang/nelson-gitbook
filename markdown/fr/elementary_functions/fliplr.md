# fliplr

Inverser l'ordre des éléments de gauche à droite

## Syntaxe

- B = fliplr(A)

## Argument d'entrée

- A - un tableau

## Argument de sortie

- B - tableau inversé.

## Description

<p>
            fliplr renvoie un nouveau tableau de A inversé de gauche à droite.</p>

## Exemple

```matlab
x = eye(3, 2);
y = fliplr(x)
```
