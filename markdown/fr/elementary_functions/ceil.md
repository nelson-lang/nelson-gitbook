# ceil

Arrondir vers le haut

## Syntaxe

- C = ceil(A)

## Argument d'entrée

- A - une variable

## Argument de sortie

- C - résultat de ceil.

## Description

<p>
            ceil renvoie une matrice d'entiers ou complexes dont les éléments sont arrondis vers le haut.</p>

## Exemple

```matlab
ceil(pi)
```
