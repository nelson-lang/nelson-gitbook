# bin2dec

Convertit un nombre en base 2 en décimal.

## Syntaxe

- D = bin2dec(TXT)

## Argument d'entrée

- TXT - un tableau de caractères.

## Argument de sortie

- D - résultat de bin2dec : une valeur entière.

## Description

<p>
            bin2dec convertit un nombre en base 2 en décimal.</p>

<p>Remarque : bin2dec et dec2bin sont des opérations réciproques.</p>

## Exemple

```matlab
bin2dec('11')
```
