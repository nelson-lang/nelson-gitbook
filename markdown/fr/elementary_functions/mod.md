# mod

Modulus after division.

## Syntaxe

- C = mod(A, B)

## Argument d'entr√©e

- A - a variable: dividend
- B - a variable: divisor

## Argument de sortie

- C - result of mod(A, B)

## Description

<p>
            C = mod(A, B) computes the modulo of A and B, i.e : A - B .* floor (A ./ B).</p>

<p>This function manages also negative values.</p>

## Exemple

```matlab
 mod (-1, 3)
```
