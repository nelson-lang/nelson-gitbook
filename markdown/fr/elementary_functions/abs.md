# abs

Valeur absolue

## Syntaxe

- R = abs(M)

## Argument d'entrée

- M - une variable

## Argument de sortie

- R - résultat de abs : valeur absolue.

## Description

<p>
            abs calcule la valeur absolue.</p>

<p>Si l'argument est un nombre complexe, abs calcule la magnitude complexe.</p>

## Exemple

```matlab
x = [1+i,-i;i,2i];
r = abs(x)
```
