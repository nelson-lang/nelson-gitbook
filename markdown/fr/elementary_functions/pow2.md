# pow2

Base 2 exponentiation and scaling of floating-point numbers.

## Syntaxe

- Y = pow2(E)
- Y = pow2(X, E)

## Argument d'entr√©e

- E - Exponent values
- X - Significand values

## Argument de sortie

- Y - result of pow2.

## Description

<p>
            Y = pow2(E) computes 2 to the power of E.</p>

<p>
                Y = pow2(X, E) computes X times 2 to the power of E.</p>

## Exemple

```matlab
R = pow2([1, 2, 3; 4, 5, 6], [6, 5, 4; 3, 2, 1])
```
