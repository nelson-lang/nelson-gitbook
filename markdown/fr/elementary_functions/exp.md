# exp

Exponentielle

## Syntaxe

- R = exp(M)

## Argument d'entrée

- M - une variable

## Argument de sortie

- R - résultat de exp : exponentielle.

## Description

<p>
            exp calcule la fonction exponentielle.</p>

<p>Pour les nombres réels :</p>

$$e^x$$

<p>Pour les nombres complexes z = x + iy :</p>

$$e^z = e^x(\cos y + i\sin y)$$

## Exemple

```matlab
x = [1+i,-i;i,2i];
r = exp(x)
```
