# conj

Conjugué complexe

## Syntaxe

- CZ = conj(M)

## Argument d'entrée

- M - une variable

## Argument de sortie

- CZ - résultat de conj : conjugué complexe.

## Description

<p>
            conj renvoie le conjugué complexe.</p>

## Exemple

```matlab
x = [1+i,-i;i,2i];
r = conj(x)
```
