# fix

Arrondir vers zéro

## Syntaxe

- C = fix(A)

## Argument d'entrée

- A - une variable

## Argument de sortie

- C - résultat de fix.

## Description

<p>
            fix renvoie une matrice d'entiers obtenue en arrondissant chaque élément vers zéro.</p>

## Exemple

```matlab
fix(pi)
```
