# hadamard

Matrice de Hadamard

## Syntaxe

- H = hadamard(n)
- H = hadamard(n, classname)

## Argument d'entrée

- n - entier scalaire : ordre.
- classname - vecteur de caractères ou chaîne scalaire : nom de la classe désirée ('double' par défaut).

## Argument de sortie

- H - Matrice de Hadamard.

## Description

<p>
            H = hadamard(n) renvoie la matrice de Hadamard d'ordre n.</p>

## Bibliographie

https://en.wikipedia.org/wiki/Hadamard_matrix , https://mathworld.wolfram.com/HadamardMatrix.html

## Exemple

```matlab
H = hadamard(4)
```
