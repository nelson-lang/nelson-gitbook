# factorial

Fonction factorielle

## 📝 Syntaxe

- R = factorial(M)

## 📥 Argument d'entrée

- M - un entier, matrice réelle single ou double.

## 📤 Argument de sortie

- R - résultat de la fonction factorielle.

## 📄 Description

<b>factorial</b> calcule la fonction factorielle : le produit des entiers 1 \* 2 \* ... \* M.

## 💡 Exemple

```matlab
R = factorial([1:10])
R = factorial(int8(4))
```
