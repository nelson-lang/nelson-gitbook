# factorial

Fonction factorielle

## Syntaxe

- R = factorial(M)

## Argument d'entrée

- M - un entier, matrice réelle single ou double.

## Argument de sortie

- R - résultat de la fonction factorielle.

## Description

<p>
            factorial calcule la fonction factorielle : le produit des entiers 1 * 2 * ... * M.</p>

## Exemple

```matlab
R = factorial([1:10])
R = factorial(int8(4))
```
