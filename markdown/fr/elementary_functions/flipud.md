# flipud

Inverser l'ordre des éléments de haut en bas

## Syntaxe

- B = flipud(A)

## Argument d'entrée

- A - un tableau

## Argument de sortie

- B - tableau inversé.

## Description

<p>
            flipud renvoie un nouveau tableau de A inversé de haut en bas.</p>

## Exemple

```matlab
x = eye(3, 2);
y = flipud(x)
```
