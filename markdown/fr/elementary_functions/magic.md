# magic

Magic square

## Syntaxe

- M = magic(N)

## Argument d'entr√©e

- N - Matrix order, specified as a scalar integer.

## Argument de sortie

- M - result of magic function.

## Description

<p>
            M = magic(N) computes an square matrix constructed as an arrangement of the 1:n^2 such that the row sums, column sums, and diagonal sums are all equal to the same value.</p>

## Exemple

```matlab
M = magic(3)
```
