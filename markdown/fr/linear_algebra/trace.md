# trace

Trace d'une matrice.

## Syntaxe

- res = trace(x)

## Argument d'entrée

- x - une valeur numérique : scalaire ou matrice (double ou simple précision)

## Argument de sortie

- res - une valeur numérique : un scalaire

## Description

<p>
                        trace(x) calcule la trace de x, la somme des éléments sur la diagonale principale.</p>

## Exemple

```matlab
X = [1 0; 0 3];
Y = trace(X)
```
