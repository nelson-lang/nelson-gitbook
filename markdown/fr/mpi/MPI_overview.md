#

Accès aux fonctionnalités MPI depuis Nelson.

## Description

<p>Comme beaucoup d'applications MPI, MPI/Nelson doit être lancée via la commande mpiexec/mpirun.</p>

<p>Les fonctionnalités MPI sont uniquement disponibles en mode CLI. Toutefois, vous pouvez les invoquer depuis d'autres modes en utilisant le builtin mpiexec.</p>
