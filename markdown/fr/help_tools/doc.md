# doc

Affiche la documentation.

## Syntaxe

- doc
- doc function_name
- doc('function_name')

## Argument d'entrée

- function_name - a string: function name

## Description

<p>doc lance le navigateur d'aide.</p>

<p>doc('function_name') affiche l'aide de la fonction indiquée par 'function_name'.</p>

## Exemples

```matlab
doc()
```

```matlab
doc sin
```

```matlab
doc is
```
