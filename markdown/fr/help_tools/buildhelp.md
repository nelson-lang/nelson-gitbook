# buildhelp

Génère l'aide des modules de Nelson.

## 📝 Syntaxe

- buildhelp()
- buildhelp(module_name)

## 📥 Argument d'entrée

- module_name - une chaîne : nom du module (le module doit être chargé).

## 📄 Description

<b>buildhelp</b> génère les fichiers d'aide.

## 💡 Exemple

```matlab
buildhelp();
buildhelp('core');
```
