# buildhelp

Génère l'aide des modules de Nelson.

## Syntaxe

- buildhelp()
- buildhelp(module_name)

## Argument d'entrée

- module_name - a string: module name (module must be loaded).

## Description

<p>buildhelp génère les fichiers d'aide.</p>

## Exemple

```matlab
buildhelp();
buildhelp('core');
```
