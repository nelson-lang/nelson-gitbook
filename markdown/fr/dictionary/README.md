# Dictionnaires

Le module Dictionnaire fournit des outils pour travailler avec des paires clé-valeur dans Nelson.

Il permet la création et la configuration de dictionnaires avec des types définis pour les clés et les valeurs, la consultation et la modification des entrées, ainsi que la gestion de la structure globale.

Ce module permet un stockage, une récupération et une manipulation efficaces des données indexées par des clés uniques, ce qui le rend adapté aux tableaux associatifs, recherches et gestion dynamique des données.

## Functions

- [configureDictionary](configureDictionary.md) - Génère un dictionnaire avec des types définis pour les clés et les valeurs.
- [dictionary](dictionary.md) - Objet qui associe des clés uniques à des valeurs.
- [entries](entries.md) - Paires clé-valeur du dictionnaire.
- [insert](insert.md) - Ajouter des entrées à un dictionnaire.
- [isConfigured](isConfigured.md) - Vérifie si le dictionnaire a des types assignés aux clés et aux valeurs.
- [isKey](isKey.md) - Vérifie si le dictionnaire contient la clé
- [keyHash](keyHash.md) - Créer un code de hachage pour une clé de dictionnaire.
- [keyMatch](keyMatch.md) - Vérifie si deux clés de dictionnaire sont identiques.
- [keys](keys.md) - Clés du dictionnaire.
- [lookup](lookup.md) - Trouver la valeur dans le dictionnaire par clé.
- [numEntries](numEntries.md) - Nombre de paires clé-valeur dans le dictionnaire.
- [remove](remove.md) - Supprimer des entrées du dictionnaire.
- [types](types.md) - Types des clés et valeurs du dictionnaire.
- [values](values.md) - Valeurs du dictionnaire.
