# beep

Produit un son de bip.

## Syntaxe

- beep
- beep(str)
- str = beep

## Argument d'entrée

- str - une chaîne : 'on' ou 'off'.

## Argument de sortie

- str - une chaîne : 'on' ou 'off'.

## Description

        beep produit un son de bip système.

## Exemple

```matlab
beep('off')
beep
beep('on')
beep
s = beep
```
