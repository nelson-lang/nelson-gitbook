# audiorecorder_pause

Met en pause un objet audiorecorder.

## 📝 Syntaxe

- pause(recObj)

## 📥 Argument d'entrée

- recObj - un objet audiorecorder.

## 📄 Description

<b>pause</b> met en pause un objet audiorecorder.

## 💡 Exemple

```matlab
recObj = audiorecorder();
      resume(recObj);
pause(recObj)

```
