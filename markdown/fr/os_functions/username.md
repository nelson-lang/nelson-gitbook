# username

obtenir le nom d'utilisateur courant.

## Syntaxe

- s = username()

## Argument de sortie

- s - un tableau de caractères : nom d'utilisateur.

## Description

<p>username renvoie le nom d'utilisateur actuellement utilisé.</p>

## Exemple

```matlab
username()
```
