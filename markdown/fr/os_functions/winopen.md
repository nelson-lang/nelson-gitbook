# winopen

Ouvrir un fichier dans l'application appropriée (Windows seulement).

## Syntaxe

- winopen(filename)

## Argument d'entrée

- command - une chaîne : commande à exécuter dans le shell.

## Description

<p>winopen(filename) ouvre le fichier dans l'application Microsoft Windows appropriée.</p>

<p>La fonction winopen utilise la commande shell Windows appropriée et effectue la même action que si vous double-cliquiez sur le fichier dans l'Explorateur Windows.</p>

<p>Si filename n'est pas dans le répertoire courant, spécifiez le chemin absolu.</p>
