# hostname

obtenir le nom d'hôte de cet ordinateur.

## Syntaxe

- s = hostname()

## Argument de sortie

- s - un tableau de caractères : nom d'hôte.

## Description

<p>hostname renvoie le nom d'hôte de cet ordinateur.</p>

## Exemple

```matlab
hostname()
```
