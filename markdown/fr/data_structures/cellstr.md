# cellstr

Convertit en tableau cellulaire de chaînes de caractères.

## Syntaxe

- ce = cellstr(A)

## Argument d'entrée

- A - une chaîne, un tableau de chaînes, ou un tableau cellulaire de tableaux de caractères.

## Argument de sortie

- ce - un tableau cellulaire de tableaux de caractères

## Description

<p>
            cellstr(A) convertit en tableau cellulaire de tableaux de caractères.</p>

## Exemples

```matlab
cellstr('Nelson')
```

```matlab
cellstr({'Nelson'})
```

```matlab
cellstr({})
```
