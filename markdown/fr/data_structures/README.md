# Structures de données

Le module Structures de données fournit des outils pour créer, manipuler et inspecter des tableaux, cellules et structures dans Nelson.

Il permet la conversion entre différents formats de données, l'accès et la modification des champs, l'application de fonctions aux éléments de tableaux et l'organisation des données structurées.

Ce module facilite la gestion flexible des données complexes, en prenant en charge les opérations programmatiques et la gestion dynamique des données.

## Functions

- [arrayfun](arrayfun.md) - Appliquer une fonction à chaque élément d'un tableau.
- [cell](cell.md) - Créer un tableau cellulaire de matrices vides.
- [cell2mat](cell2mat.md) - Transformer un tableau cellulaire contenant des matrices en une seule matrice concaténée.
- [cell2struct](cell2struct.md) - Créer un struct à partir d'un tableau cellulaire.
- [celldisp](celldisp.md) - Afficher le contenu d'un tableau cellulaire.
- [cellfun](cellfun.md) - Évalue une fonction sur un tableau cellulaire.
- [cellstr](cellstr.md) - Convertit en tableau cellulaire de chaînes de caractères.
- [fieldnames](fieldnames.md) - Renvoie les noms de champs d'une structure ou d'un handle.
- [getfield](getfield.md) - Renvoie la valeur d'un champ dans un struct.
- [iscellstr](iscellstr.md) - Renvoie si une variable est un tableau cellulaire de chaînes.
- [isfield](isfield.md) - Vérifie si un nom de champ existe dans une structure.
- [namedargs2cell](namedargs2cell.md) - Convertit une structure contenant des paires nom-valeur en un tableau cellulaire.
- [num2cell](num2cell.md) - Convertir un tableau en tableau cellulaire avec des cellules de tailles cohérentes.
- [orderfields](orderfields.md) - Réorganiser les champs d'un tableau de structures.
- [renameStructField](renameStructField.md) - Renommer les noms de champs d'un struct ou d'un tableau de structs.
- [rmfield](rmfield.md) - Supprimer des champs d'une structure.
- [setfield](setfield.md) - Définir le contenu d'un champ de structure.
- [struct](struct.md) - Créer un struct.
- [struct2cell](struct2cell.md) - Créer un tableau cellulaire à partir d'une structure.
