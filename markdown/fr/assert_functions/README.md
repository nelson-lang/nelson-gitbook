# Fonctions d'assertion

Le module Fonctions d'assertion de Nelson fournit des outils pour valider le comportement des programmes et vérifier les conditions lors du développement ou des tests.

Ces fonctions permettent de s'assurer que les calculs et conditions logiques respectent les critères attendus, améliorant la fiabilité du code et détectant les erreurs précocement.

## Functions

- [assert](assert.md) - Vérifie que la condition est vraie.
- [assert_checkerror](assert_checkerror.md) - Vérifie qu'une commande lève l'erreur attendue.
- [assert_isapprox](assert_isapprox.md) - Vérifie que les valeurs calculées et attendues sont approximativement égales.
- [assert_isequal](assert_isequal.md) - Vérifie que les valeurs calculées et attendues sont égales.
- [assert_isfalse](assert_isfalse.md) - Vérifie que la condition est fausse.
- [assert_istrue](assert_istrue.md) - Vérifie que la condition est vraie.
