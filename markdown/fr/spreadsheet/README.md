# Feuille de calcul

Le module Feuille de calcul fournit des fonctions pour lire et écrire des données tabulaires depuis et vers des formats de feuille de calcul basés sur du texte, tels que CSV et les fichiers séparés par délimiteurs.

Il prend en charge l'importation vers différents types de données comme les tableaux numériques, les cellules de chaînes et les tables, ainsi que leur exportation vers des fichiers.

Cela permet une interaction fluide avec les logiciels de tableur (Excel, LibreOffice Calc, etc.) et l'échange de données entre applications.

## Functions

- [csvread](csvread.md) - Lire un fichier de valeurs séparées par des virgules (CSV).
- [csvwrite](csvwrite.md) - Écrire un fichier de valeurs séparées par des virgules (CSV).
- [detectImportOptions](detectImportOptions.md) - Créer des options d'importation basées sur le contenu du fichier.
- [dlmread](dlmread.md) - Lire une matrice numérique depuis un fichier texte utilisant un délimiteur.
- [dlmwrite](dlmwrite.md) - Écrire une matrice numérique dans un fichier texte en utilisant un délimiteur.
- [readcell](readcell.md) - Créer une cellule à partir d'un fichier.
- [readmatrix](readmatrix.md) - Créer une matrice à partir d'un fichier.
- [readtable](readtable.md) - Créer une table à partir d'un fichier.
- [writecell](writecell.md) - Écrire un tableau de cellules dans un fichier.
- [writematrix](writematrix.md) - Écrire une matrice dans un fichier.
- [writetable](writetable.md) - Écrire une table dans un fichier.
