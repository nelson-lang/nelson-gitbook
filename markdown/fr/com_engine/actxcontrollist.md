# actxcontrollist

Obtenir les contrôles ActiveX disponibles.

## Syntaxe

- l = actxcontrollist()

## Argument de sortie

- l - une cellule de chaînes 1 par 3.

## Description

<p>
            l = actxcontrollist() retourne la liste des contrôles disponibles</p>

## Exemple

```matlab
l = actxcontrollist()
```
