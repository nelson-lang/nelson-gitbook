# Interface client du modèle d'objet composant (COM)

Le module COM Engine permet à Nelson d'interagir avec les composants Windows COM (Component Object Model) et ActiveX.

Il permet l'automatisation des applications logicielles, l'accès à leurs méthodes et propriétés, et la manipulation d'objets COM directement depuis les scripts Nelson.

Cela fournit un moyen d'intégrer Nelson avec une large gamme de logiciels Windows, y compris les applications Microsoft Office.

## Functions

- [Exemples COM](1_COM_examples.md) - Quelques exemples pour montrer comment utiliser l'interface COM.
- [COM_delete](COM_delete.md) - Supprime un contrôle ou serveur COM.
- [COM_fieldnames](COM_fieldnames.md) - Retourne les noms des propriétés d'un objet COM.
- [COM_get](COM_get.md) - Obtenir la valeur d'une propriété depuis l'interface COM.
- [COM_invoke](COM_invoke.md) - Invoquer une méthode sur un objet ou interface COM.
- [COM_ismethod](COM_ismethod.md) - Détermine si l'entrée est une méthode existante d'un objet COM.
- [COM_isprop](COM_isprop.md) - Détermine si l'entrée est une propriété existante d'un objet COM.
- [COM_methods](COM_methods.md) - Retourne les noms des méthodes d'un objet COM.
- [COM_range](COM_range.md) - Fonction privée : assistant de plage.
- [COM_set](COM_set.md) - Définir la propriété d'un objet ou interface à la valeur spécifiée.
- [COM_used](COM_used.md) - Retourne la liste des handles COM actuellement utilisés.
- [COM_xlsfinfo](COM_xlsfinfo.md) - Détermine si le fichier contient une feuille de calcul Microsoft Excel.
- [COM_xlsread](COM_xlsread.md) - Lire un fichier de feuille de calcul Microsoft Excel en utilisant COM.
- [COM_xlswrite](COM_xlswrite.md) - Écrire un fichier de feuille de calcul Microsoft Excel en utilisant COM.
- [actxGetRunningServer](actxGetRunningServer.md) - Handle vers une instance en cours d'exécution du serveur Automation.
- [actxcontrollist](actxcontrollist.md) - Obtenir les contrôles ActiveX disponibles.
- [actxserver](actxserver.md) - Crée un serveur COM.
- [actxserverlist](actxserverlist.md) - Obtenir les serveurs ActiveX disponibles.
- [iscom](iscom.md) - Détermine si l'entrée est un objet COM ou ActiveX.
