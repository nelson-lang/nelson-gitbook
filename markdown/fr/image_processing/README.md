# Fonctions de traitement d'image

Le module Image Processing fournit des opérations de base pour manipuler des images, y compris la conversion de format, le redimensionnement et la rotation.

## Functions

- [im2double](im2double.md) - Convertit une image en précision double.
- [imresize](imresize.md) - Redimensionne une image par échelle ou taille de sortie
- [imrotate](imrotate.md) - Fait pivoter une image d'un angle spécifié
