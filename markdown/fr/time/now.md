# now

Renvoie la date et l'heure courantes sous forme de numéro de date série.

## Syntaxe

- n = now()

## Argument de sortie

- n - un double.

## Description

<p>
            now() renvoie la date et l'heure courantes sous forme d'un numéro de date série.</p>

## Exemple

```matlab
datevec(now())
```
