# date

Retourne la date courante sous forme de vecteur de caractères.

## Syntaxe

- d = date()

## Argument de sortie

- d - une chaîne : date au format dd-MMM-yyyy. MMM : abréviation anglaise du mois.

## Description

<p>d = date() renvoie la date courante sous forme de vecteur de caractères au format dd-MMM-yyyy.</p>

## Exemple

```matlab
d = date()
fix(c)
```
