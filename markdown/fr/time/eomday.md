# eomday

Returns last day of month.

## Syntaxe

- E = eomday(Y, M)

## Argument d'entrée

- Y - année : réel.
- M - mois : réel.

## Argument de sortie

- E - dernier jour du mois : réel.

## Description

<p>
            E = eomday(Y, M) returns the last day of the month M for the year Y.</p>

## Exemple

```matlab
eomday(1900, 1:12)
```
