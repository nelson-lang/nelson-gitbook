# plus

Addition, opérateur +

## Syntaxe

- C = plus(A, B)
- C = A + B

## Argument d'entrée

- A - une variable
- B - une variable

## Argument de sortie

- C - résultat de A + B

## Description

<p>
            C = plus(A, B) effectue l'addition A + B.</p>

## Exemples

```matlab
plus(3, 4)
3 + 4
```

```matlab
[1, 2] + 1
plus([1, 2], 1)
```

```matlab
ones(0, 0) + 1
```
