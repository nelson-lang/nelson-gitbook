# mtimes

Multiplication élément par élément, opérateur .\*

## Syntaxe

- C = times(A, B)
- C = A .\* B

## Argument d'entrée

- A - une variable
- B - une variable

## Argument de sortie

- C - résultat de A .\* B

## Description

<p>
            C = times(A, B) effectue la multiplication élément par élément : A .* B.</p>

## Exemples

```matlab
times(3, 4)
3 .* 4
```

```matlab
M1 = [2 6 10; 4 8 70];
M2 = [-25 88 1; 23 29 41];
M1 .* M2
```
