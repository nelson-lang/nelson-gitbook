# mrdivide

Division matricielle à droite, opérateur /.

## Syntaxe

- C = mrdivide(A, B)
- C = A / B

## Argument d'entrée

- A - une variable
- B - une variable

## Argument de sortie

- C - résultat de A / B

## Description

<p>C = mrdivide(A, B) retourne la division matricielle à droite de A par B.</p>

## Exemple

```matlab
B = ones(3, 4)
A = B *2
A / B

```
