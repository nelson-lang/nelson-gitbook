# uplus

Unaire plus, opérateur +

## Syntaxe

- C = uplus(A)
- C = +A

## Argument d'entrée

- A - une variable

## Argument de sortie

- C - résultat de +A

## Description

<p>
            C = uplus(A) effectue l'opération unaire plus, c.-à-d. +A.</p>

## Exemple

```matlab
M = -3;
+M
```
