# ne

Inégalité, opérateur ~=

## Syntaxe

- C = ne(A, B)
- C = A ~= B

## Argument d'entrée

- A - une variable
- B - une variable

## Argument de sortie

- C - résultat de A ~= B

## Description

<p>
            C = ne(A, B) effectue l'opération d'inégalité : A ~= B.</p>

<p>
                ne compare les parties réelle et imaginaire des tableaux numériques.</p>

## Exemple

```matlab
ne(3, 4)
3 ~= 4
```
