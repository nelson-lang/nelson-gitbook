# uminus

Unaire moins, opérateur -

## Syntaxe

- C = uminus(A)
- C = -A

## Argument d'entrée

- A - une variable

## Argument de sortie

- C - résultat de -A

## Description

<p>
            C = uminus(A) effectue l'opération unaire moins, c.-à-d. -A.</p>

## Exemple

```matlab
M = 3;
 -M
```
