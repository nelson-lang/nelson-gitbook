# subsindex

Convertir un objet en vecteur d'indices.

## Syntaxe

- r = subsindex(O)

## Argument d'entrée

- O - une variable

## Description

<p>Si O est un objet alors subsindex est la méthode de surcharge qui permet de convertir cet objet en un vecteur d'indexation valide.</p>
