# minus

Soustraction, opérateur -

## Syntaxe

- C = minus(A, B)
- C = A - B

## Argument d'entrée

- A - une variable
- B - une variable

## Argument de sortie

- C - résultat de A - B

## Description

<p>
            C = minus(A, B) effectue la soustraction A - B.</p>

## Exemples

```matlab
minus(3, 4)
3 - 4
```

```matlab
[1, 2] - 1
minus([1, 2], 1)
```

```matlab
ones(0, 0) - 1
```
