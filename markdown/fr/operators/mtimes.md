# mtimes

Multiplication matricielle, opérateur \*

## Syntaxe

- C = mtimes(A, B)
- C = A \* B

## Argument d'entrée

- A - une variable
- B - une variable

## Argument de sortie

- C - résultat de A \* B

## Description

<p>
        C = mtimes(A, B) effectue l'opération de multiplication matricielle : A * B.</p>

## Exemples

```matlab
mtimes(3, 4)
3 * 4
```

```matlab
M1 = [2 6 10; 4 8 70];
M2 = [-25 88 1; 23 29 41; 24 40 0];
M1 * M2
```
