# not

négation logique, opérateur ~

## 📝 Syntaxe

- C = not(A)
- C = ~A

## 📥 Argument d'entrée

- A - une variable

## 📤 Argument de sortie

- C - résultat de ~A

## 📄 Description

<b>C = not(A)</b> effectue la négation logique : ~A.

## 💡 Exemple

```matlab
M = false(3, 3);
~M
```
