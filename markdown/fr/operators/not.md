# not

négation logique, opérateur ~

## Syntaxe

- C = not(A)
- C = ~A

## Argument d'entrée

- A - une variable

## Argument de sortie

- C - résultat de ~A

## Description

<p>
            C = not(A) effectue la négation logique : ~A.</p>

## Exemple

```matlab
M = false(3, 3);
~M
```
