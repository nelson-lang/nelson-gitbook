# mpower

Puissance matricielle, opérateur ^

## Syntaxe

- C = mpower(A, B)
- C = A ^ B

## Argument d'entrée

- A - une variable
- B - une variable

## Argument de sortie

- C - résultat de A^B

## Description

<p>
            C = mpower(A, B) effectue l'opération de puissance matricielle : A^B</p>

## Exemple

```matlab
mpower(3, 4)
3^4
```
