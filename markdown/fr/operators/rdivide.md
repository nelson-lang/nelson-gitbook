# rdivide

Division droite, opérateur ./

## Syntaxe

- C = rdivide(A, B)
- C = A ./ B

## Argument d'entrée

- A - une variable
- B - une variable

## Argument de sortie

- C - résultat de A ./ B

## Description

<p>
            C = rdivide(A, B) effectue la division élément par élément à droite : A ./ B.</p>

## Exemples

```matlab
rdivide(3, 4)
3 ./ 4
```

```matlab
M1 = [2];
M2 = [-25 88 1];
M1 ./ M2
```
