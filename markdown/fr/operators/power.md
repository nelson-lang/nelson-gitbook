# power

Puissance élément par élément, opérateur .^

## Syntaxe

- C = power(A, B)
- C = A .^ B

## Argument d'entrée

- A - une variable
- B - une variable

## Argument de sortie

- C - résultat de A.^B

## Description

<p>
        C = power(A, B) effectue une opération de puissance élément par élément : A .^ B.</p>

## Exemple

```matlab
power(3, 4)
3.^4
```
