# isprop

Retourne vrai si c'est une propriété d'un objet handle.

## Syntaxe

- res = isprop(h, propertyname)

## Argument d'entrée

- h - un objet handle
- propertyname - une chaîne

## Argument de sortie

- res - un booléen : vrai ou faux

## Description

        isprop renvoie vrai si l'argument est un handle valide avec le nom de propriété, et faux sinon.
