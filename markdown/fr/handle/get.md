# get

Récupère la valeur d'une propriété d'un objet handle.

## Syntaxe

- R = get(h, property_name)

## Argument d'entrée

- h - un objet handle.
- property_name - une chaîne : nom de la propriété.

## Argument de sortie

- R - Le type de donnée de la valeur renvoyée dépend de la méthode invoquée.

## Description

<p>R = get(h, property_name) renvoie la valeur de la propriété demandée.</p>
