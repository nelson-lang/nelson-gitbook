# set

Définit la valeur d'une propriété d'un objet handle.

## Syntaxe

- R = set(h, property_name, value)

## Argument d'entrée

- h - un objet handle.
- property_name - une chaîne : nom de la propriété.
- value - une variable.

## Argument de sortie

- R - propriétés modifiables par l'utilisateur et valeurs possibles pour l'objet identifié par h.

## Description

<p>Cette routine peut être utilisée pour modifier la valeur d'une propriété spécifiée d'un objet handle.</p>
