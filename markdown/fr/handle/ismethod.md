# ismethod

Retourne vrai si c'est une méthode d'un objet handle.

## Syntaxe

- res = ismethod(h, methodname)

## Argument d'entrée

- h - un objet handle
- methodname - une chaîne

## Argument de sortie

- res - un booléen : vrai ou faux

## Description

        ismethod renvoie vrai si l'argument est un handle valide avec le nom de méthode, et faux sinon.
