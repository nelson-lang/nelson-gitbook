# methods

Renvoie les noms des méthodes d'un objet handle.

## Syntaxe

- c = methods(h)

## Argument d'entrée

- h - un objet handle

## Argument de sortie

- c - un tableau (cell) de chaînes

## Description

        methods renvoie un tableau de chaînes contenant les noms des méthodes.
