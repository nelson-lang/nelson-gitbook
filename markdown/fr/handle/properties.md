# properties

Renvoie les noms des propriétés d'un objet handle.

## Syntaxe

- properties(h)
- c = properties(h)

## Argument d'entrée

- h - un objet handle

## Argument de sortie

- c - un tableau (cell) de chaînes

## Description

        properties renvoie un tableau de chaînes contenant les noms des propriétés.
