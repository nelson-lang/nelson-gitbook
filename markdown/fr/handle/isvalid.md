# isvalid

Retourne vrai si la variable h est un handle valide.

## Syntaxe

- res = isvalid(h)

## Argument d'entrée

- h - un objet handle

## Argument de sortie

- res - un booléen : vrai ou faux

## Description

        isvalid renvoie vrai si l'argument est un handle valide, et faux sinon.
