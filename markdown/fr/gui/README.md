# Module GUI

Le module GUI fournit des fonctions pour créer et interagir avec les composants d'interface graphique, les boîtes de dialogue et les fenêtres d'application.

## Functions

- [commandhistory](commandhistory.md) - Historique des commandes
- [errordlg](errordlg.md) - Crée une boîte de dialogue d'erreur.
- [filebrowser](filebrowser.md) - Explorateur du dossier courant
- [helpdlg](helpdlg.md) - Crée une boîte de dialogue d'aide.
- [inserthtml](inserthtml.md) - Insère du HTML dans la console GUI.
- [lookandfeel](lookandfeel.md) - Apparence et comportement (look and feel) de l'application
- [msgbox](msgbox.md) - Crée une boîte de dialogue de message.
- [openvar](openvar.md) - Ouvre une variable dans l'éditeur de variables
- [qt_verbose](qt_verbose.md) - Afficher/masquer les messages de débogage Qt.
- [questdlg](questdlg.md) - Crée une boîte de dialogue de question.
- [uigetdir](uigetdir.md) - Ouvre une boîte de dialogue pour sélectionner un répertoire.
- [warndlg](warndlg.md) - Crée une boîte de dialogue d'avertissement.
- [workspace](workspace.md) - Explorateur d'espace de travail
