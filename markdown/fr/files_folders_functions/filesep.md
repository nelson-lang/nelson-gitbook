# filesep

Renvoie le caractère séparateur de fichiers pour la plateforme courante.

## Syntaxe

- res = filesep()

## Argument de sortie

- res - a string: '/' ou '\\'

## Description

filesep renvoie '\\' sur Windows et '/' sur les autres plateformes.

## Exemple

```matlab
A = filesep
```
