# tempdir

Renvoie le chemin du répertoire temporaire.

## Syntaxe

- tempdir()
- p = tempdir()

## Argument de sortie

- p - a string: répertoire temporaire courant.

## Description

<p>Renvoie le nom du répertoire de fichiers temporaires du système hôte.</p>

## Exemple

```matlab
r = tempdir()
```
