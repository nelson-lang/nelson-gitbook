# pwd

Renvoie le répertoire courant.

## Syntaxe

- pwd()
- r = pwd()

## Argument de sortie

- r - a string: répertoire courant.

## Description

<p>Renvoie le répertoire de travail courant.</p>

<p>pwd() sans argument affiche le répertoire courant.</p>

<p></p>

## Exemple

```matlab
r = pwd()
pwd()
```
