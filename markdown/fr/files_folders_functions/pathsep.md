# pathsep

Renvoie le caractère séparateur de chemins pour la plateforme courante.

## Syntaxe

- res = pathsep()

## Argument de sortie

- res - a string: ';' ou ':'

## Description

pathsep renvoie ';' sur Windows et ':' sur les autres plateformes.

## Exemple

```matlab
A = pathsep
```
