# userdir

Renvoie le chemin du répertoire utilisateur courant.

## Syntaxe

- userdir()
- p = userdir()

## Argument de sortie

- p - a string: répertoire utilisateur courant.

## Description

<p>Renvoie le nom du répertoire de l'utilisateur.</p>

## Exemple

```matlab
r = userdir()
```
