# smartindent

Indentation intelligente d'un fichier Nelson

## Syntaxe

- smartindent(filename)
- smartindent(filename, indentsize)
- smartindent(filename, indentsize, dobackup)

## Argument d'entrée

- filename - une chaîne : nom du fichier à indenter.
- indentsize - un entier > 0. par défaut : 2
- dobackup - un booléen : false par défaut

## Description

        smartindent indente le code pour faciliter la lecture d'instructions telles que les boucles while.
