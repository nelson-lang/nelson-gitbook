# db2mag

Convertit un gain en décibels (dB) en magnitude.

## Syntaxe

- mag = db2mag(db)

## Argument d'entrée

- db - tableau d'entrée : scalaire, vecteur ou matrice.

## Argument de sortie

- mag - magnitude correspondante

## Description

<p>
            mag = db2mag(db) renvoie la magnitude correspondante.</p>

## Exemple

```matlab
mag = db2mag([0, -20])
```
