# sinc

Fonction sinc.

## Syntaxe

- c = sinc(m)

## Argument d'entrée

- m - tableau d'entrée : scalaire, vecteur ou matrice.

## Argument de sortie

- c - sinc de l'entrée

## Description

<p>
        c = sinc(m) renvoie un tableau c dont les éléments sont le sinc des éléments de l'entrée : m.</p>

## Exemple

```matlab
c = sinc(pi)
```
