# restoredefaultpath

Restaure le chemin de Nelson à son état initial au démarrage.

## Syntaxe

- restoredefaultpath

## Description

<p>
            restoredefaultpath restaure le chemin de recherche de Nelson à son état de démarrage.</p>

## Exemple

```matlab
path
path('')
path
restoredefaultpath
path
```
