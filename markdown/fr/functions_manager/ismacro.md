# ismacro

Vérifie l'existence d'une macro (fonction).

## Syntaxe

- tf = ismacro(name)

## Argument d'entrée

- name - une chaîne : nom de macro.

## Argument de sortie

- tf - un booléen : vrai si la macro existe.

## Description

<p>
            ismacro vérifie l'existence d'une macro.</p>

## Exemple

```matlab
ismacro('isbuiltin')
ismacro('exist')
```
