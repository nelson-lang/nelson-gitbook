# rehash

Réinitialise le cache des répertoires du chemin de recherche de Nelson.

## Syntaxe

- rehash

## Description

<p>
            rehash() réinitialise le cache des répertoires du chemin de recherche de Nelson.</p>

<p>Cela se produit chaque fois que Nelson affiche l'invite.</p>

<p>Vous devriez utiliser rehash() uniquement lorsque vous exécutez un fichier .m qui met à jour un autre fichier .m</p>

## Exemple

```matlab
rehash()
```
