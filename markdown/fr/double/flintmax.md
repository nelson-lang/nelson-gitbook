# flintmax

Plus grand entier consécutif représentable en virgule flottante.

## Syntaxe

- R = flintmax()
- R = flintmax('double')
- R = flintmax('single')
- R = flintmax('like', V)

## Argument d'entrée

- V - une variable double ou single.

## Argument de sortie

- R - un double ou single.

## Description

<p>
            flintmax renvoie le plus grand entier consécutif représentable au format virgule flottante.</p>

## Exemple

```matlab
flintmax
flintmax('double')
flintmax('like', pi)
flintmax('single')
```
