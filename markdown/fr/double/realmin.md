# realmin

Plus petit nombre flottant positif.

## Syntaxe

- R = realmin()
- R = realmin('double')
- R = realmin('single')

## Argument de sortie

- R - un double ou single.

## Description

<p>
            realmin renvoie le plus petit nombre flottant positif.</p>

## Exemple

```matlab
realmin
realmin('double')
realmin('single')
```
