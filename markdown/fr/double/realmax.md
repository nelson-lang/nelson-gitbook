# realmax

Plus grand nombre flottant positif.

## Syntaxe

- R = realmax()
- R = realmax('double')
- R = realmax('single')

## Argument de sortie

- R - un double ou single.

## Description

<p>
            realmax renvoie le plus grand nombre flottant positif.</p>

## Exemple

```matlab
realmax
realmax('double')
realmax('single')
```
