# single

Convertit une variable en type simple précision.

## Syntaxe

- S = single(V)

## Argument d'entrée

- V - une variable.

## Argument de sortie

- S - une variable simple précision.

## Description

<p>
            single(V) convertit vers le type simple précision.</p>

## Exemples

```matlab
single('Nelson')
```

```matlab
A = single(pi)
```
