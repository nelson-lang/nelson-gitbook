# atan

Calcule la tangente inverse en radians pour chaque élément de x.

## Syntaxe

- res = atan(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        atan calcule la tangente inverse en radians pour chaque élément de x.

## Exemple

```matlab
A = eye(3, 3);
res = atan(A)
```
