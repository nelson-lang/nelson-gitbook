# atan2d

Tangente inverse à quatre quadrants en degrés.

## 📝 Syntaxe

- d = atan2d(y, x)

## 📥 Argument d'entrée

- y - une valeur numérique
- x - une valeur numérique

## 📤 Argument de sortie

- d - a numeric value

## 📄 Description

<b>d = atan2d(y, x)</b> retourne la tangente inverse à quatre quadrants (tan-1) de <b>y</b> et <b>x</b>, qui doivent être réels.

## 💡 Exemple

```matlab
x = [1 0 -1 0];
y = [0 1 0 -1];
d = atan2d(y, x)
```
