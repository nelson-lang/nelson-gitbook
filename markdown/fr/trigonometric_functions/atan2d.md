# atan2d

Tangente inverse à quatre quadrants en degrés.

## Syntaxe

- d = atan2d(y, x)

## Argument d'entrée

- y - une valeur numérique
- x - une valeur numérique

## Argument de sortie

- d - a numeric value

## Description

<p>
            d = atan2d(y, x) retourne la tangente inverse à quatre quadrants (tan-1) de y et x, qui doivent être réels.</p>

## Exemple

```matlab
x = [1 0 -1 0];
y = [0 1 0 -1];
d = atan2d(y, x)
```
