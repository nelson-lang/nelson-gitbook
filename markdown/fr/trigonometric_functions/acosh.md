# acosh

Cosinus hyperbolique inverse.

## Syntaxe

- res = acosh(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        acosh calcule le cosinus hyperbolique inverse.

## Exemple

```matlab
A =  [1+2i, 2, -3];
res = acosh(A)
```
