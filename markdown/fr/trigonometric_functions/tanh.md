# tanh

Calcule la tangente hyperbolique en radians pour chaque élément de x.

## Syntaxe

- res = tanh(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        tanh calcule la tangente hyperbolique en radians pour chaque élément de x.

## Exemple

```matlab
A = eye(3, 3);
res = tanh(A)
```
