# asec

Sécante inverse d'un angle en radians.

## 📝 Syntaxe

- res = asec(x)

## 📥 Argument d'entrée

- x - une valeur numérique

## 📤 Argument de sortie

- res - une valeur numérique

## 📄 Description

<b>asec</b> calcule la sécante inverse de l'argument en radians pour chaque élément de <b>x</b>.

## 💡 Exemple

```matlab
x = -pi:0.75:pi;
R = asec(x)
```
