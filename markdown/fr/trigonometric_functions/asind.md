# asind

Sinus inverse en degrés.

## Syntaxe

- res = asind(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        asind calcule le sinus inverse en degrés pour chaque élément de x.

## Exemple

```matlab
x = [-50 -20 0 20 50];
y = asind(x)
```
