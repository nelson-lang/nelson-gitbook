# acotd

Cotangente inverse d'un angle en degrés

## Syntaxe

- res = acotd(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        acotd calcule la cotangente inverse d'un angle pour chaque élément de x en degrés.

## Exemple

```matlab
R = acotd([-i pi+i*pi/2 -1+i*4])
```
