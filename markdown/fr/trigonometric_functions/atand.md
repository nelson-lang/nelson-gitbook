# atand

Tangente inverse en degrés.

## Syntaxe

- res = atand(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        atand calcule la tangente inverse en degrés pour chaque élément de x.

## Exemple

```matlab
x = [-50 -20 0 20 50];
y = atand(x)
```
