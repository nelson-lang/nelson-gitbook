# sind

Calcule le sinus en degrés pour chaque élément de x.

## Syntaxe

- res = sind(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        sind calcule le sinus en degrés pour chaque élément de x.

## Exemple

```matlab
A = [0 30 45 60 90 360];
sind(A)
```
