# rad2deg

Convertit un angle de radians en degrés.

## Syntaxe

- d = rad2deg(r)

## Argument d'entrée

- r - une valeur numérique (double ou simple)

## Argument de sortie

- d - une valeur numérique

## Description

        d = rad2deg(r) convertit les unités d'angle de radians en degrés pour chaque élément de r.

## Exemple

```matlab
dist = 7194;
radEarth = 6371;
D = rad2deg(dist / radEarth)
```
