# cscd

Cosécante de l'argument en degrés.

## Syntaxe

- res = cscd(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        cscd calcule la cosécante de l'argument en degrés pour chaque élément de x.

## Exemple

```matlab
R = cscd([35+i 15+2i 10+3i])
```
