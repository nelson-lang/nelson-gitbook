# cscd

Cosécante de l'argument en degrés.

## 📝 Syntaxe

- res = cscd(x)

## 📥 Argument d'entrée

- x - une valeur numérique

## 📤 Argument de sortie

- res - une valeur numérique

## 📄 Description

<b>cscd</b> calcule la cosécante de l'argument en degrés pour chaque élément de <b>x</b>.

## 💡 Exemple

```matlab
R = cscd([35+i 15+2i 10+3i])
```
