# csch

Cosécante hyperbolique.

## Syntaxe

- res = csch(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        csch calcule la cosécante hyperbolique pour chaque élément de x.

## Exemple

```matlab
X = [3*pi, 2*pi, pi, 0];
R = csch(X)
```
