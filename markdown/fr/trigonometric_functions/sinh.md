# sinh

Calcule le sinus hyperbolique en radians pour chaque élément de x.

## Syntaxe

- res = sinh(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        sinh calcule le sinus hyperbolique en radians pour chaque élément de x.

## Exemple

```matlab
A = eye(3, 3);
res = sinh(A)
```
