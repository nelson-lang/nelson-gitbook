# acos

Calcule le cosinus inverse en radians pour chaque élément de x.

## Syntaxe

- res = acos(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

acos calcule le cosinus inverse en radians pour chaque élément de x.

## Exemple

```matlab
A = eye(3, 3);
res = acos(A)

```
