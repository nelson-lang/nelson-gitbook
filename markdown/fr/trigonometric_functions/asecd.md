# asecd

Sécante inverse de l'argument en degrés.

## Syntaxe

- res = asecd(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        asecd calcule la sécante inverse de l'argument en degrés pour chaque élément de x.

## Exemple

```matlab
R = asecd([1, 10+3i, 15+2i, 35+i])
```
