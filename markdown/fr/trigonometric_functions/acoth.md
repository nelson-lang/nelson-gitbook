# acoth

Cotangente hyperbolique inverse.

## Syntaxe

- res = acoth(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        acoth calcule la cotangente hyperbolique inverse.

## Exemple

```matlab
A =  [1+2i, 2, -3];
res = acoth(A)
```
