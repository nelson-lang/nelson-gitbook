# cosd

Calcule le cosinus en degrés pour chaque élément de x.

## Syntaxe

- res = cosd(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        cosd calcule le cosinus en degrés pour chaque élément de x.

## Exemple

```matlab
A = [0 30 45 60 90 360];;
res = cosd(A)
```
