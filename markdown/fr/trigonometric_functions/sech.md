# sech

Sécante hyperbolique.

## Syntaxe

- res = sech(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        sech calcule la sécante hyperbolique pour chaque élément de x.

## Exemple

```matlab
X = [3*pi, 2*pi, pi, 0];
R = sech(X)
```
