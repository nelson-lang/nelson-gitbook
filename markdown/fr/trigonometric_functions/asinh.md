# asinh

Sinus hyperbolique inverse

## Syntaxe

- res = asinh(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        asinh calcule le sinus hyperbolique inverse en radians pour chaque élément de x.

## Exemple

```matlab
A = eye(3, 3);
res = asinh(A)
```
