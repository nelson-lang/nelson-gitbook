# coth

Cotangente hyperbolique.

## 📝 Syntaxe

- res = coth(x)

## 📥 Argument d'entrée

- x - une valeur numérique

## 📤 Argument de sortie

- res - une valeur numérique

## 📄 Description

<b>coth</b> calcule la cotangente hyperbolique pour chaque élément de <b>x</b>.

## 💡 Exemple

```matlab
X = [3*pi, 2*pi, pi, 0];
R = coth(X)
```
