# cot

Cotangente d'un angle en radians

## 📝 Syntaxe

- res = cot(x)

## 📥 Argument d'entrée

- x - une valeur numérique

## 📤 Argument de sortie

- res - une valeur numérique

## 📄 Description

<b>cot</b> calcule la cotangente d'un angle pour chaque élément de <b>x</b>.

## 💡 Exemple

```matlab
R = cot([-i pi+i*pi/2 -1+i*4])
```
