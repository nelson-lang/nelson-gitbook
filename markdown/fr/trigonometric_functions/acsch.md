# acsch

Cosécante hyperbolique inverse.

## 📝 Syntaxe

- res = acsch(x)

## 📥 Argument d'entrée

- x - une valeur numérique

## 📤 Argument de sortie

- res - une valeur numérique

## 📄 Description

<b>acsch</b> calcule la cosécante hyperbolique inverse pour chaque élément de <b>x</b>.

## 💡 Exemple

```matlab
X = [3*pi, 2*pi, pi, 0];
R = acsch(X)
```
