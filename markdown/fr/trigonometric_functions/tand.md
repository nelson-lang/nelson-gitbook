# tand

Calcule la tangente en degrés pour chaque élément de x.

## Syntaxe

- res = tand(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        tand calcule la tangente en degrés pour chaque élément de x.

## Exemple

```matlab
A = [0 30 45 60 90 360];
res = tand(A)
```
