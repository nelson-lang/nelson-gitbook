# secd

Sécante de l'argument en degrés.

## 📝 Syntaxe

- res = secd(x)

## 📥 Argument d'entrée

- x - une valeur numérique

## 📤 Argument de sortie

- res - une valeur numérique

## 📄 Description

<b>secd</b> calcule la sécante de l'argument en degrés pour chaque élément de <b>x</b>.

## 💡 Exemple

```matlab
R = secd([1, 10+3i, 15+2i, 35+i])
```
