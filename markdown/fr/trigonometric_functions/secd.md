# secd

Sécante de l'argument en degrés.

## Syntaxe

- res = secd(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        secd calcule la sécante de l'argument en degrés pour chaque élément de x.

## Exemple

```matlab
R = secd([1, 10+3i, 15+2i, 35+i])
```
