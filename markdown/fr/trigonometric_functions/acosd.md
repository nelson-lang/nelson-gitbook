# acosd

Cosinus inverse en degrés.

## 📝 Syntaxe

- res = acosd(x)

## 📥 Argument d'entrée

- x - une valeur numérique

## 📤 Argument de sortie

- res - une valeur numérique

## 📄 Description

<b>acosd</b> calcule le cosinus inverse en degrés pour chaque élément de <b>x</b>.

## 💡 Exemple

```matlab
x = [1 -20 0 2 5];
y = acosd(x)
```
