# asin

Calcule le sinus inverse en radians pour chaque élément de x.

## Syntaxe

- res = asin(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        asin calcule le sinus inverse en radians pour chaque élément de x.

## Exemple

```matlab
A = eye(3, 3);
res = asin(A)
```
