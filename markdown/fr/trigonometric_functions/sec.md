# sec

Sécante d'un angle en radians.

## Syntaxe

- res = sec(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        sec calcule la sécante de l'argument en radians pour chaque élément de x.

## Exemple

```matlab
x = -pi:0.75:pi;
R = sec(x)
```
