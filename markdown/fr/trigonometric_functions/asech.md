# asech

Sécante hyperbolique inverse d'un angle en radians.

## 📝 Syntaxe

- res = asech(x)

## 📥 Argument d'entrée

- x - une valeur numérique

## 📤 Argument de sortie

- res - une valeur numérique

## 📄 Description

<b>asech</b> calcule la sécante hyperbolique inverse de l'argument en radians pour chaque élément de <b>x</b>.

## 💡 Exemple

```matlab
x = -pi:0.75:pi;
R = asech(x)
```
