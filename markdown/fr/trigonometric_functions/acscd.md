# acscd

Cosécante inverse en degrés.

## 📝 Syntaxe

- res = acsc(x)

## 📥 Argument d'entrée

- x - une valeur numérique

## 📤 Argument de sortie

- res - une valeur numérique

## 📄 Description

<b>acscd</b> calcule la cosécante inverse de l'argument en degrés pour chaque élément de <b>x</b>.

## 💡 Exemple

```matlab
x = [0 1 20 10 Inf];
y = acscd(x)
```
