# acscd

Cosécante inverse en degrés.

## Syntaxe

- res = acsc(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        acscd calcule la cosécante inverse de l'argument en degrés pour chaque élément de x.

## Exemple

```matlab
x = [0 1 20 10 Inf];
y = acscd(x)
```
