# acot

Cotangente inverse d'un angle en radians

## Syntaxe

- res = acot(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        acot calcule la cotangente inverse d'un angle pour chaque élément de x.

## Exemple

```matlab
R = acot([-i pi+i*pi/2 -1+i*4])
```
