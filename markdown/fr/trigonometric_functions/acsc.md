# acsc

Cosécante inverse en radians.

## Syntaxe

- res = acsc(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        acsc calcule la cosécante inverse de l'argument en radians pour chaque élément de x.

## Exemple

```matlab
R = acsc(3)
R = acsc(0.5)
```
