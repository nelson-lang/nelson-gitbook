# acsc

Cosécante inverse en radians.

## 📝 Syntaxe

- res = acsc(x)

## 📥 Argument d'entrée

- x - une valeur numérique

## 📤 Argument de sortie

- res - une valeur numérique

## 📄 Description

<b>acsc</b> calcule la cosécante inverse de l'argument en radians pour chaque élément de <b>x</b>.

## 💡 Exemple

```matlab
R = acsc(3)
R = acsc(0.5)
```
