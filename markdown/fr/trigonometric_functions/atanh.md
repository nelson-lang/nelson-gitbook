# atanh

Calcule la tangente hyperbolique inverse.

## Syntaxe

- res = atanh(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        atanh calcule la tangente hyperbolique inverse.

## Exemple

```matlab
A =  [1+2i, 2, -3];
res = atanh(A)
```
