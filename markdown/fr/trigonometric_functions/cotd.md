# cotd

Cotangente de l'argument en degrés

## 📝 Syntaxe

- res = cotd(x)

## 📥 Argument d'entrée

- x - une valeur numérique

## 📤 Argument de sortie

- res - une valeur numérique

## 📄 Description

<b>cotd</b> calcule la cotangente de l'argument en degrés pour chaque élément de <b>x</b>.

## 💡 Exemple

```matlab
R = cotd(35 + 5i)
```
