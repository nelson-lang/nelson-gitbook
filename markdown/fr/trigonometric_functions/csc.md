# csc

Cosécante d'un angle en radians.

## Syntaxe

- res = csc(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        csc calcule la cosécante de l'argument en radians pour chaque élément de x.

## Exemple

```matlab
R = csc(-pi+0.01:0.01:-0.01)
```
