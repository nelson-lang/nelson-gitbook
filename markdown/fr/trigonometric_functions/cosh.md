# cosh

Calcule le cosinus hyperbolique en radians pour chaque élément de x.

## Syntaxe

- res = cosh(x)

## Argument d'entrée

- x - une valeur numérique

## Argument de sortie

- res - une valeur numérique

## Description

        cosh calcule le cosinus hyperbolique en radians pour chaque élément de x.

## Exemple

```matlab
A = eye(3, 3);
res = cosh(A)
```
