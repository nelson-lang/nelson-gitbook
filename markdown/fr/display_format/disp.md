# disp

Afficher une variable.

## Syntaxe

- disp(V)

## Argument d'entrée

- V - une variable

## Description

<p>
            disp(V) affiche la valeur de la variable V.</p>

<p>
                disp utilise le réglage courant de format pour afficher les valeurs numériques.</p>

## Exemples

```matlab
disp('Hello Nelson')
```

```matlab
disp(pi)
```

```matlab
disp(eye(3, 3))
```

disp always ends with a newline.

```matlab
disp('')
```
