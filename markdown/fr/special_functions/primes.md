# primes

Nombres premiers inférieurs ou égaux à la valeur d'entrée

## Syntaxe

- p = primes(n)

## Argument d'entrée

- n - scalaire, valeur entière réelle

## Argument de sortie

- p - vecteur avec les nombres premiers.

## Description

<p>
            p = primes(n) retourne un vecteur ligne contenant tous les nombres premiers inférieurs ou égaux à n.</p>

<p>Le type de données de p est le même que celui de n.</p>

## Exemple

```matlab
p = primes(15)
```
