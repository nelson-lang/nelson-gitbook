# Fonctions spéciales

Le module Fonctions Spéciales fournit des outils pour effectuer des opérations mathématiques avancées dans Nelson.

Il inclut des fonctions pour les distributions statistiques, les calculs combinatoires, et d'autres calculs mathématiques spécialisés qui sont essentiels dans diverses applications scientifiques et d'ingénierie.

Ce module améliore les capacités de Nelson en offrant une gamme de fonctions qui supportent des analyses complexes et des tâches de modélisation.

## Functions

- [betainc](betainc.md) - Fonction bêta incomplète
- [cross](cross.md) - Produit vectoriel.
- [dot](dot.md) - Produit scalaire.
- [erf](erf.md) - Fonction d'erreur
- [erfc](erfc.md) - Fonction d'erreur complémentaire
- [erfcinv](erfcinv.md) - Fonction d'erreur complémentaire inverse
- [erfcx](erfcx.md) - Fonction d'erreur complémentaire mise à l'échelle
- [erfinv](erfinv.md) - Fonction d'erreur inverse
- [factor](factor.md) - Facteurs premiers
- [gamma](gamma.md) - Fonction spéciale gamma
- [gammaln](gammaln.md) - Logarithme de la fonction gamma
- [gcd](gcd.md) - Plus grand commun diviseur
- [interp1](interp1.md) - Interpolation de donnees 1-D
- [interp2](interp2.md) - Interpolation de donnees grillees 2-D au format meshgrid
- [interp3](interp3.md) - Interpolation de donnees grillees 3-D au format meshgrid
- [interpn](interpn.md) - Interpolation de donnees grillees N-D au format ndgrid
- [peaks](peaks.md) - Fonction peaks
- [primes](primes.md) - Nombres premiers inférieurs ou égaux à la valeur d'entrée
