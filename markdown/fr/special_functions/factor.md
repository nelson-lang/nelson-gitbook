# factor

Facteurs premiers

## Syntaxe

- f = factor(n)

## Argument d'entrée

- n - scalaire entier réel, non négatif

## Argument de sortie

- p - vecteur avec les facteurs premiers.

## Description

<p>
            f = factor(n) retourne un vecteur ligne avec les facteurs premiers de n.</p>

<p>Le vecteur f est du même type de données que n.</p>

## Exemple

```matlab
f = factor(204)
```
