# dllib_used

Renvoie la liste des handles dllib actuellement utilisés

## Syntaxe

- r = dllib_used()

## Argument de sortie

- h - un vecteur de handles dllib.

## Description

<p>Renvoie la liste des handles dllib actuellement utilisés.</p>

## Exemple

```matlab
dllib_used(),delete(dllib_used())
```
