# removecompilerconf

Supprime la configuration du compilateur utilisée (sous Windows)

## Syntaxe

- res = removecompilerconf()

## Argument de sortie

- res - a logical

## Description

<p>
            removecompilerconf renvoie true si un compilateur avait été configuré avec configuremsvc ou configuremingw.</p>

<p>
                removecompilerconf renvoie toujours true sur les autres plateformes.</p>
