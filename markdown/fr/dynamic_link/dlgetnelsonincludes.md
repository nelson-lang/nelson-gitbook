# dlgetnelsonincludes

Renvoie les chemins des répertoires d'includes de Nelson

## Syntaxe

- C = dlgetnelsonincludes()

## Argument de sortie

- C - un tableau de cellules contenant les chemins des répertoires d'includes utilisés par les modules Nelson

## Description

<p>
        C = dlgetnelsonincludes() renvoie un tableau de cellules contenant les chemins des répertoires d'includes utilisés par les modules Nelson.</p>

<p>Ces chemins sont utilisés en interne pour le développement des modules et les processus de compilation.</p>

## Exemple

See module skeleton for example

```matlab
dlgetnelsonincludes()
```
