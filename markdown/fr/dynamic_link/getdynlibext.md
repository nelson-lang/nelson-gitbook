# getdynlibext

Renvoie l'extension des bibliothèques dynamiques

## 📝 Syntaxe

- ext = getdynlibext()

## 📤 Argument de sortie

- ext - une chaîne : extension des bibliothèques dynamiques

## 📄 Description

<b>getdynlibext()</b> returns the extension of dynamic libraries.

## 💡 Exemple

```matlab
getdynlibext()
```
