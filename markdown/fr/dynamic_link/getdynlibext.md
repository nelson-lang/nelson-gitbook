# getdynlibext

Renvoie l'extension des bibliothèques dynamiques

## Syntaxe

- ext = getdynlibext()

## Argument de sortie

- ext - une chaîne : extension des bibliothèques dynamiques

## Description

<p>
            getdynlibext() returns the extension of dynamic libraries.</p>

## Exemple

```matlab
getdynlibext()
```
