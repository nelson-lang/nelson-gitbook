# vswhere

Localiser les installations de Visual Studio (2017, 2019 et versions ultérieures)

## Syntaxe

- res = vswhere()

## Argument de sortie

- res - a struct with information about Visual studio

## Description

<p>
            vswhere permet de trouver facilement Visual Studio.</p>

<p>
                vswhere est actuellement implémenté seulement sur la plateforme Windows.</p>

## Bibliographie

https://github.com/Microsoft/vswhere

## Exemple

```matlab
vswhere()
```
