# libpointer_used

Renvoie la liste des handles libpointer actuellement utilisés

## Syntaxe

- r = libpointer_used()

## Argument de sortie

- h - a vector of libpointer handle.

## Description

<p>Renvoie la liste des handles libpointer actuellement utilisés.</p>

## Exemple

```matlab
libpointer_used(),delete(libpointer_used())
```
