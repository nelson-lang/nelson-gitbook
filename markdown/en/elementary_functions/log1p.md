# log1p

log(1 + x) accurately for small values of x.

## Syntax

- R = log1p(M)

## Input argument

- M - a variable

## Output argument

- R - result of log(1 + x) accurately for small values of x.

## Description

  <p><b>log1p</b> computes log(1 + x) accurately for small values of x.</p>

## Example

```matlab
x = [1+i,-i;i,2i];
r = log1p(x)
```
