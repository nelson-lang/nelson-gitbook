# deal

Distribute inputs to outputs.

## Syntax

- [R1, ... , Rn] = deal(A1, ... , An)
- [R1, ... , Rn] = deal(A)

## Input argument

- A1, ... , An - variables

## Output argument

- R1, ... , Rn - variables

## Description

  <p><b>deal</b> replicates the input parameters to the corresponding output parameters.</p>
  <p>If a singular input parameter is provided, its value will be duplicated across all outputs.</p>

## Examples

```matlab
[A1, A2, A3] = deal(pi)
```

```matlab
S = [];
S.A = [];
S(2).A = [];
S(3).A = [];
A1 = 200;
A2 = 'fifo';
A3 = 1:11;
[S.A] = deal(A1, A2, A3)
```

```matlab
C = cell(1,3)
A1 = 200;
A2 = 'fifo';
A3 = 1:11;
[C{:}] = deal(A1, A2, A3)
```
