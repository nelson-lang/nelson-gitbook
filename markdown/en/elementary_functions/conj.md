# conj

Complex conjugate

## Syntax

- CZ = conj(M)

## Input argument

- M - a variable

## Output argument

- CZ - result of conj: complex conjugate.

## Description

<p>
            <b>conj</b> returns the complex conjugate.</p>

## Example

```matlab
x = [1+i,-i;i,2i];
r = conj(x)
```
