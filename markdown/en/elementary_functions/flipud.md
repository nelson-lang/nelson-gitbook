# flipud

Flip order of elements up to dow

## Syntax

- B = flipud(A)

## Input argument

- A - an array

## Output argument

- B - flipped array.

## Description

<p>
            fliplr return an new array of A flipped up to down.</p>

## Example

```matlab
x = eye(3, 2);
y = flipud(x)
```
