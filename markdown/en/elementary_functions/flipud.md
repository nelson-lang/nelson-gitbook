# flipud

Flip order of elements up to dow

## Syntax

- B = flipud(A)

## Input argument

- A - an array

## Output argument

- B - flipped array.

## Description

  <p><b>fliplr</b> return an new array of <b>A</b> flipped up to down.</p>

## Example

```matlab
x = eye(3, 2);
y = flipud(x)
```
