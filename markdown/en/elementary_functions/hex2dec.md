# hex2dec

Convert number in base 16 to decimal.

## Syntax

- D = hex2dec(TXT)

## Input argument

- TXT - a char array.

## Output argument

- D - result of hex2dec: an integer value.

## Description

  <p><b>hex2dec</b> converts number in base 16 to decimal.</p>
  <p>Note:</p>
  <p> - <b>hex2dec</b> and <b>dec2hex</b> are inverses of one another.</p>

## Example

```matlab
bin2dec('11')
```
