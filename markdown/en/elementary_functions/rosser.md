# rosser

Classic symmetric eigenvalue test problem.

## Syntax

- R = rosser()
- R = rosser(classname)

## Input argument

- classname - row character vector or scalar string: class name desired ('double' by default).

## Output argument

- R - Rosser Matrix.

## Description

<p>
            R = rosser() returns the Rosser Matrix.</p>

```matlab
R = rosser()
```
