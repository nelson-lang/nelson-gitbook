# minus

Subtraction, - operator

## Syntax

- C = minus(A, B)
- C = A - B

## Input argument

- A - a variable
- B - a variable

## Output argument

- C - result of A - B

## Description

  <p><b>C = minus(A, B)</b> performs subtraction A - B variables.</p>

## Examples

```matlab
minus(3, 4)
3 - 4
```

```matlab
[1, 2] - 1
minus([1, 2], 1)
```

```matlab
ones(0, 0) - 1
```
