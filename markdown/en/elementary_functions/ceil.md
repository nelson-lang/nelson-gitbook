# ceil

Round up

## Syntax

- C = ceil(A)

## Input argument

- A - a variable

## Output argument

- C - result of ceil.

## Description

  <p><b>ceil</b> returns an integer or complex matrix made of rounded up elements.</p>

## Example

```matlab
ceil(pi)
```
