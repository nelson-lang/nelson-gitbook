# imag

Imaginary part of an complex number.

## Syntax

- im = imag(M)

## Input argument

- M - a variable

## Output argument

- R - imaginary part of the elements of the complex array M.

## Description

<p>
            R = imag(M) Return the imaginary part of M.</p>

## Example

```matlab
cplx = 22+34*i;
r = imag(cplx)
```
