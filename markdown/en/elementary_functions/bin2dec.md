# bin2dec

Convert number in base 2 to decimal.

## Syntax

- D = bin2dec(TXT)

## Input argument

- TXT - a char array.

## Output argument

- D - result of bin2dec: an integer value.

## Description

<p>
            bin2dec converts number in base 2 to decimal.</p>

<p>Note:</p>

<p> - bin2dec and dec2bin are inverses of one another.</p>

## Example

```matlab
bin2dec('11')
```
