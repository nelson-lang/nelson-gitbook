# fix

Round towards zero

## Syntax

- C = fix(A)

## Input argument

- A - a variable

## Output argument

- C - result of fix.

## Description

<p>
            fix returns an integer matrix made of nearest rounded integers toward zeros.</p>

## Example

```matlab
fix(pi)
```
