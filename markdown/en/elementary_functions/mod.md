# mod

Modulus after division.

## Syntax

- C = mod(A, B)

## Input argument

- A - a variable: dividend
- B - a variable: divisor

## Output argument

- C - result of mod(A, B)

## Description

<p>
            <b>C = mod(A, B)</b> computes the modulo of A and B, i.e : A - B .* floor (A ./ B).</p>
<p>This function manages also negative values.</p>

## Example

```matlab
 mod (-1, 3)
```
