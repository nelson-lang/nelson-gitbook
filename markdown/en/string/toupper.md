# toupper

Upper case conversion.

## Syntax

- res = toupper(str)

## Input argument

- str - a row character array, a cell of strings or an string array.

## Output argument

- res - a string upper case

## Description

<b>toupper</b> converts a string to upper case.

## Examples

```matlab
toupper('NelSon')
```

```matlab
upper(["NelSon", "is", "open"])
```
