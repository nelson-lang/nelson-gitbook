# buildhelp

Build help of Nelson's modules.

## 📝 Syntax

- buildhelp()
- buildhelp(module_name)

## 📥 Input argument

- module_name - a string: module name (module must be loaded).

## 📄 Description

<b>buildhelp</b> generates help files.

## 💡 Example

```matlab
buildhelp();
buildhelp('core');
```
