# siogetvariable

Emit an value of variable to web browser.

## Syntax

- siogetvariable(variable_name)

## Input argument

- variable_name - a string: name of variable

## Description

  <p><b>siogetvariable</b> send value of a variable to web browser.</p>

## Example

```matlab
A = eye(3, 3); siogetvariable('A')
```
