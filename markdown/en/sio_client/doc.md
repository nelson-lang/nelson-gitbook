# doc

Displays documentation.

## Syntax

- doc
- doc function_name
- doc('function_name')

## Input argument

- function_name - a string: function name

## Description

  <p><b>doc</b>launches web browser.</p>
  <p><b>doc('function_name')</b>displays the help about function designed by 'function_name'.</p>

## Examples

```matlab
doc()
```

```matlab
doc sin
```

```matlab
doc is
```
