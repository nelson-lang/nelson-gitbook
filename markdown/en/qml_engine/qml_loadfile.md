# qml_loadfile

Load a QML file.

## Syntax

- h = qml_loadfile(filename)

## Input argument

- filename - a string: a QML filename.

## Output argument

- h - a QObject handle.

## Description

<p>Load a QML file</p>

<p>It creates a QML component and load .qml file.</p>
