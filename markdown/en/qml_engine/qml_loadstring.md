# qml_loadstring

Load a QML string.

## Syntax

- h = qml_loadstring(str_to_eval)

## Input argument

- str_to_eval - a string.

## Output argument

- h - a QObject handle.

## Description

<p>Load a QML string</p>
<p>It creates a QML component and load .qml file.</p>
