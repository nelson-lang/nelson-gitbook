# nelsonObject

nelson object callable from QML.

## Syntax

- nelson.disp(msg)
- nelson.evaluate(cmd)
- nelson.processevent()
- nelson.call(function_name)
- nelson.call(function_name, arg1, ..., arg5)

## Input argument

- msg - a string.
- cmd - a string.
- function_name - a string: nelson function name to call
- arg1, ..., arg5 - javascript variables

## Description

<p>
            nelson object contains some methods used as callback to call nelson from QML</p>
