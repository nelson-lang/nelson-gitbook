# qml_createqquickview

Load a QML file and creates a window.

## Syntax

- h = qml_createqquickview(filename)

## Input argument

- filename - a string: a QML filename.

## Output argument

- h - a QObject handle.

## Description

  <p>Load a QML file</p>
  <p>It creates a QML component, a window, and load .qml file.</p>
