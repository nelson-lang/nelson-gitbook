# QObject_root

QObject root object.

## Syntax

- r = QObject_root()

## Output argument

- h - QObject handle of Nelson gui.

## Description

<p>Returns QObject handle of Nelson gui.</p>

## Example

```matlab
h1 = QObject_root()
h1.windowTitle
h1.windowTitle = 'Your title'
```
