# pow2db

Convert power to decibel.

## Syntax

- db = pow2db(pow)

## Input argument

- pow - input array: scalar, vector or matrix.

## Output argument

- db - corresponding values in decibels

## Description

<p>
            db = pow2db(pow) returns corresponding values in decibels.</p>

## Example

```matlab
DB = pow2db([1, 0.01])
```
