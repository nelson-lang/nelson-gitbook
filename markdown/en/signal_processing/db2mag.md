# db2mag

Convert a gain in decibels (dB) to a magnitude.

## Syntax

- mag = db2mag(db)

## Input argument

- db - input array: scalar, vector or matrix.

## Output argument

- mag - corresponding magnitude

## Description

<p>
            <b>mag = db2mag(db)</b> returns corresponding magnitude.</p>

## Example

```matlab
mag = db2mag([0, -20])
```
