# mag2db

Convert a magnitude to decibels (dB).

## Syntax

- db = mag2db(mag)

## Input argument

- mag - input array: scalar, vector or matrix.

## Output argument

- db - corresponding values in decibels

## Description

<p>
            <b>db = mag2db(mag)</b> returns corresponding values in decibels.</p>

## Example

```matlab
DB = mag2db([1, 0.01])
```
