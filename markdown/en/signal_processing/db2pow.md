# db2pow

Convert a gain in decibels (dB) to power.

## Syntax

- pow = db2pow(db)

## Input argument

- db - input array: scalar, vector or matrix.

## Output argument

- pow - corresponding power

## Description

  <p><b>pow = db2pow(db)</b> returns corresponding power.</p>

## Example

```matlab
pow = db2pow([0, -20])
```
