# rdivide

Right division, ./ operator

## Syntax

- C = rdivide(A, B)
- C = A ./ B

## Input argument

- A - a variable
- B - a variable

## Output argument

- C - result of A ./ B

## Description

  <p><b>C = rdivide(A, B)</b> performs right division operation: A ./* B.</p>

## Examples

```matlab
rdivide(3, 4)
3 ./ 4
```

```matlab
M1 = [2];
M2 = [-25 88 1];
M1 ./ M2
```
