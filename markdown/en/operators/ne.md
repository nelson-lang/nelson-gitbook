# ne

Inequality, ~= operator

## Syntax

- C = ne(A, B)
- C = A ~= B

## Input argument

- A - a variable
- B - a variable

## Output argument

- C - result of A ~= B

## Description

<p>
            C = ne(A, B) performs inequality operation: A ~= B variables.</p>

<p>
                ne compares both real and imaginary parts of numeric arrays.</p>

## Example

```matlab
ne(3, 4)
3 ~= 4
```
