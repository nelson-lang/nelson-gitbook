# subsindex

Convert an object to an index vector.

## Syntax

- r = subsindex(O)

## Input argument

- O - a variable

## Description

  <p>If <b>O</b> is an object then <b>subsindex</b> is the overloading method that allows to convert this object to a valid indexing vector.</p>
