# not

not logical, ~ operator

## Syntax

- C = not(A)
- C = ~A

## Input argument

- A - a variable

## Output argument

- C - result of ~A

## Description

<p>
            C = not(A) performs not logical ~A.</p>

## Example

```matlab
M = false(3, 3);
~M
```
