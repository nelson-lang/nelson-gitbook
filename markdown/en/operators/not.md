# not

not logical, ~ operator

## 📝 Syntax

- C = not(A)
- C = ~A

## 📥 Input argument

- A - a variable

## 📤 Output argument

- C - result of ~A

## 📄 Description

<b>C = not(A)</b> performs not logical ~A.

## 💡 Example

```matlab
M = false(3, 3);
~M
```
