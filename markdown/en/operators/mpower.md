# mpower

Matrix power, ^ operator

## Syntax

- C = mpower(A, B)
- C = A ^ B

## Input argument

- A - a variable
- B - a variable

## Output argument

- C - result of A^B

## Description

  <p><b>C = mpower(A, B)</b> performs matrix power operation: A^B</p>

## Example

```matlab
mpower(3, 4)
3^4
```
