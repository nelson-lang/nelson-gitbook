# uminus

Unary minus, - operator

## Syntax

- C = uminus(A)
- C = -A

## Input argument

- A - a variable

## Output argument

- C - result of -A

## Description

<p>
            C = uminus(A) performs unary minus ie -A.</p>

## Example

```matlab
M = 3;
-M
```
