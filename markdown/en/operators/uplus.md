# uplus

Unary plus, + operator

## Syntax

- C = uplus(A)
- C = +A

## Input argument

- A - a variable

## Output argument

- C - result of +A

## Description

<p>
            C = uplus(A) performs unary plus ie +A.</p>

## Example

```matlab
M =-3;
+M
```
