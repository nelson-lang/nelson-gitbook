# plus

Addition, + operator

## Syntax

- C = plus(A, B)
- C = A + B

## Input argument

- A - a variable
- B - a variable

## Output argument

- C - result of A + B

## Description

<p>
            C = plus(A, B) performs addition A + B variables.</p>

## Examples

```matlab
plus(3, 4)
3 + 4
```

```matlab
[1, 2] + 1
plus([1, 2], 1)
```

```matlab
ones(0, 0) + 1
```
