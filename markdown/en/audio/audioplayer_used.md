# audioplayer_used

Returns list of current used audioplayer handle.

## Syntax

- r = audioplayer_used()

## Output argument

- h - a vector of audioplayer handle.

## Description

<p>Returns list of current used audioplayer handle.</p>

```matlab
audioplayer_used(),delete(audioplayer_used())
```
