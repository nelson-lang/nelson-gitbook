# audiorecorder_pause

Pause an audiorecorder object.

## 📝 Syntax

- pause(recObj)

## 📥 Input argument

- recObj - an audiorecorder object.

## 📄 Description

<b>pause</b> pauses an audiorecorder object.

## 💡 Example

```matlab
recObj = audiorecorder();
      resume(recObj);
pause(recObj)

```
