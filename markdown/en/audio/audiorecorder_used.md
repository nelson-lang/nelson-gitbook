# audiorecorder_used

Returns list of current used audiorecorder handle.

## 📝 Syntax

- r = audiorecorder_used()

## 📤 Output argument

- h - a vector of audiorecorder handle.

## 📄 Description

Returns list of current used audiorecorder handle.

## 💡 Example

```matlab
audiorecorder_used(),delete(audiorecorder_used())
```
