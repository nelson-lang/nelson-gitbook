# isreal

Return true if all imaginary part is a zero array.

## Syntax

- res = isreal(var)

## Input argument

- var - a variable

## Output argument

- res - a logical: true or false

## Description

<b>isreal</b> returns a logical true if var is a non-complex matrix or scalar and a logical false otherwise.

## Examples

```matlab
A = 1 + 0i;
res = isreal(A)
```

```matlab
B = uint8(3);
res = isreal(B)
```

```matlab
A = single([3, i]);
res = isreal(A)
```
