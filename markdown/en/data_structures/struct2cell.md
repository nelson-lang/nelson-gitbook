# struct2cell

Creates a cell from a structure.

## Syntax

- ce = struct2cell(st)

## Input argument

- st - a structure.

## Output argument

- ce - a cell.

## Description

  <p><b>ce = struct2cell(st)</b> returns a new cell from the structure.</p>

## Example

```matlab
names = {'Pierre', 'Anna', 'Roberto'}
values =  {45, 42, 13}
st = struct ('name', names, 'age', values);
ce = struct2cell(st)
```
