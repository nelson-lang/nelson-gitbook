# rmfield

Remove fields from structure.

## Syntax

- s = rmfield(st, field)

## Input argument

- st - a structure.
- field - a string, cell of strings, or char.

## Output argument

- s - a structure without field.

## Description

  <p><b>s = rmfield(st, field)</b> removes the specified field from structure array.</p>

## Example

```matlab
example.a = 1
example.b = 'nelson'
example.c = []
rmfield(example, 'b')
```
