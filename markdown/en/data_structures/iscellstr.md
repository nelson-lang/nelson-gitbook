# iscellstr

Returns if a variable is a cell of strings.

## Syntax

- true_or_false = iscellstr(A)

## Input argument

- A - a variable

## Output argument

- true_or_false - a logical

## Description

  <p><b>iscellstr(A)</b> returns true if <b>A</b> is a cell of strings or an empty cell).</p>

## Examples

```matlab
iscellstr('Nelson')
```

```matlab
iscellstr({'Nelson'})
```

```matlab
iscellstr({})
```
