# ismacro

Check for the existence of a macro (function).

## Syntax

- tf = ismacro(name)

## Input argument

- name - a string: macro name.

## Output argument

- tf - a logical: true if macro exists.

## Description

<p>
            ismacro checks for the existence of a macro.</p>

## Example

```matlab
ismacro('isbuiltin')
ismacro('exist')
```
