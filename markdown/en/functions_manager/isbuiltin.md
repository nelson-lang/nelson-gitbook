# isbuiltin

Check for the existence of a builtin.

## Syntax

- tf = isbuiltin(name)

## Input argument

- name - a string: builtin name.

## Output argument

- tf - a logical: true if builtin exists.

## Description

  <p><b>isbuiltin</b> checks for the existence of a builtin.</p>

## Example

```matlab
isbuiltin('isbuiltin')
isbuiltin('exist')
ismacro('exist')
```
