# refresh

Redraw current figure.

## Syntax

- refresh()
- refresh(F)

## Input argument

- F - figure graphics object.

## Description

<p>
            refresh erases and redraws the current figure.</p>

<p>
                refresh(F) redraws the figure identified by F.</p>
