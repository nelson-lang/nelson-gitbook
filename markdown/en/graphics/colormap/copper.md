# copper

Copper colormap array.

## 📝 Syntax

- c = copper
- c = copper(m)

## 📥 Input argument

- m - a scalar integer value: Number of colors (256 as default value).

## 📤 Output argument

- c - Copper colormap array.

## 📄 Description

<b>copper</b> returns the colormap with copper colors.

## 💡 Example

```matlab
f = figure();
surf(peaks);
colormap('copper');
```
