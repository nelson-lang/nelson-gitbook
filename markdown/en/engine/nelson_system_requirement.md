# System Requirements

System Requirements by platforms.

## Description

<p>
            Linux:</p>

<p>
            Operating system: Ubuntu 24.04 LTS, Ubuntu 22.04 LTS, Ubuntu 20.04 LTS, Fedora,
            ArchLinux, NixOs</p>

<p>
            Processor: Any Intel or AMD x86-64 processor.</p>

<p>
            RAM: See Operating System Requirements (16 GB recommanded or more).</p>

<p>
            Storage: 1 GB for all products installation and an SSD/nvme is recommended.</p>

<p>
            Graphics: A hardware-accelerated GPU with OpenGL support and at least 1GB of
            memory is recommended.</p>

<p></p>

<p>
            Windows:</p>

<p>
            Operating system: Windows 10, 11 or Windows Server 2022.</p>

<p>
            Processor: Any Intel or AMD x86-64 processor with AVX2 instructions set (CPU
            released from 2015 onwards).</p>

<p>
            RAM: See Operating System Requirements (16 GB recommanded or more).</p>

<p>
            Storage: 1 GB for all products installation and an SSD/nvme is recommended.</p>

<p>
            Graphics: A hardware-accelerated GPU with OpenGL support and at least 1GB of
            memory is recommended.</p>

<p></p>

<p>
            MacOs:</p>

<p>
            Operating system: macOS Tahoe, macOS Sequoia, macOS Sonoma, macOS Ventura.</p>

<p>
            Processor: Any Intel processor support by Apple and any M-series chip.</p>

<p>
            Storage: 1 GB for all products installation and an SSD/nvme is recommended.</p>

<p>
            Graphics: Any Mac capable of running macOS Ventura has a GPU that can run Nelson.</p>

<p></p>
