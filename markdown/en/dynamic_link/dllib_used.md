# dllib_used

Returns list of current used dllib handle.

## Syntax

- r = dllib_used()

## Output argument

- h - a vector of dllib handle.

## Description

  <p>Returns list of current used dllib handle.</p>

## See also

[dlopen](dlopen.md).

## Example

```matlab
dllib_used(),delete(dllib_used())
```
