# removecompilerconf

Remove used compiler configuration (on Windows).

## Syntax

- res = removecompilerconf()

## Output argument

- res - a logical

## Description

<p>
            removecompilerconf returns true if compiler was previously configured with configuremsvc or configuremingw.</p>

<p>
                removecompilerconf returns always true on others platforms.</p>

## See also

[configuremsvc](../dynamic_link/configuremsvc.md), [configuremingw](../dynamic_link/configuremingw.md).
