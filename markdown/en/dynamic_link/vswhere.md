# vswhere

Locate Visual Studio 2017, 2019 and newer installations

## Syntax

- res = vswhere()

## Output argument

- res - a struct with information about Visual studio

## Description

<p>
            vswhere allows to find easily Visual studio.</p>

<p>
                vswhere is currently only implemented on Windows platform.</p>

## Bibliography

https://github.com/Microsoft/vswhere

## Example

```matlab
vswhere()
```
