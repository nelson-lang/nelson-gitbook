# dlsym_used

Returns list of current used dlsym handle.

## Syntax

- r = dlsym_used()

## Output argument

- h - a vector of dlsym handle.

## Description

<p>Returns list of current used dlsym handle.</p>

## Example

```matlab
dlsym_used(),delete(dlsym_used())
```
