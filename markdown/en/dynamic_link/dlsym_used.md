# dlsym_used

Returns list of current used dlsym handle.

## 📝 Syntax

- r = dlsym_used()

## 📤 Output argument

- h - a vector of dlsym handle.

## 📄 Description

Returns list of current used dlsym handle.

## 💡 Example

```matlab
dlsym_used(),delete(dlsym_used())
```
