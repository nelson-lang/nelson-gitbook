# cmake

call CMake tool

## Syntax

- [status, message] = cmake(varargin)

## Input argument

- varargin - command to send to CMake

## Output argument

- res - a logical: true if cmake command is successfully
- message - a string: message generated by cmake command.

## Description

<p>
            cmake used internally to generate makefile used to build C/C++ code.</p>

<p>
                cmake used by dlgeneratemake.</p>
