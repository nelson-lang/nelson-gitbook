# getdynlibext

Returns the extension of dynamic libraries.

## 📝 Syntax

- ext = getdynlibext()

## 📤 Output argument

- ext - a string: dynamic library extension

## 📄 Description

<b>getdynlibext()</b> returns the extension of dynamic libraries.

## 💡 Example

```matlab
getdynlibext()
```
