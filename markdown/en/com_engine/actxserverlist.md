# actxserverlist

Get available ActiveX servers.

## Syntax

- l = actxserverlist()

## Output argument

- l - a 1 by 3cell of strings.

## Description

<p>
            l = actxserverlist() returns list of available servers.</p>

## Example

```matlab
l = actxserverlist()
```
