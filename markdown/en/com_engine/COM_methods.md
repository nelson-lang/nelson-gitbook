# COM_methods

Returns the methods name of an COM object.

## Syntax

- l = COM_methods(h)
- l = methods(h)

## Input argument

- h - a COM object.

## Output argument

- l - a cell of strings.

## Description

<description><b>methods</b> returns a cell of strings with methods name.</description>

## Example

```matlab
e = actxserver('Excel.Application');
methods(e)
delete(e)
clear e
```
