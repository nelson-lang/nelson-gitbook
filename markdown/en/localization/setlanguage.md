# setlanguage

Changes the language used in Nelson.

## Syntax

- setlanguage(language)

## Input argument

- language - a string: 'en_US', 'fr_FR' or others by default.

## Description

<p>
            setlanguage changes the language used by Nelson and saves this changes for subsequent runs of Nelson.</p>
