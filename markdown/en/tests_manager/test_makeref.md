# test_makeref

Creates a '.ref' file for a test

## Syntax

- status = test_makeref(filename)

## Input argument

- filename - a string: filename, a test file.

## Output argument

- status - a logical: true if .ref was generated.

## Description

<p>
            <b>test_makeref</b> function creates a '.ref' file from a test file.</p>
<p>test file must have <--CHECK REF--> tag.</p>
