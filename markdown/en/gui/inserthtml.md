# inserthtml

Insert html in GUI console.

## Syntax

- inserthtml(html_txt)

## Input argument

- html_txt - a string: html text

## Description

  <p><b>inserthtml</b> inserts html code in GUI console.</p>

## Example

```matlab
inserthtml(markdown(fileread([nelsonroot(),'/CHANGELOG.md'])))
```
