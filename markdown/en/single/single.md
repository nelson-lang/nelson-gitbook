# single

Converts a variable to single precision type.

## Syntax

- S = single(V)

## Input argument

- V - a variable.

## Output argument

- S - a single.

## Description

<p>
            single(V) converts to the single-precision type.</p>

## Examples

```matlab
single('Nelson')
```

```matlab
A = single(pi)
```
