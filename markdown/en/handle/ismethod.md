# ismethod

Return true if method of handle object.

## Syntax

- res = ismethod(h, methodname)

## Input argument

- h - a handle object
- methodname - a string

## Output argument

- res - a logical: true or false

## Description

<b>ismethod</b> returns a logical 1 if the argument is a valid handle with method name and a logical 0 otherwise.
