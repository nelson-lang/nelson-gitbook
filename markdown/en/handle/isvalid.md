# isvalid

Return true if variable h is a valid handle.

## Syntax

- res = isvalid(h)

## Input argument

- h - an handle object

## Output argument

- res - a logical: true or false

## Description

<b>isvalid</b> returns a logical 1 if the argument is a valid handle and a logical 0 otherwise.
