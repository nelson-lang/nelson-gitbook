# methods

Returns the methods name of an object handle.

## Syntax

- c = methods(h)

## Input argument

- h - a handle object

## Output argument

- c - a cell of strings

## Description

        methods returns a cell of strings with methods name.

## See also

[isprop](../handle/isprop.md), [ismethod](../handle/ismethod.md).
