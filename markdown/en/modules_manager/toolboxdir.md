# toolboxdir

Returns path of a module.

## Syntax

- p = toolboxdir(module_short_name)

## Input argument

- module_short_name - a string: short module's name.

## Output argument

- p - a string: path of the module.

## Description

<p>
            <b>toolboxdir</b> is an helper's function to return module root path.</p>

## Example

```matlab
toolboxdir('core')
```
