# prefdir

Return the preferences directory used by Nelson.

## Syntax

- pref_path = prefdir

## Output argument

- pref_path - a string: the preferences directory

## Description

<p>
            pref_path = prefdir() returns the preferences directory used by Nelson.</p>

## Example

an example

```matlab
cd(prefdir)
```
