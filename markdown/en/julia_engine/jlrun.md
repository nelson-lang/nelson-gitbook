# jlrun

Run Julia statements from Nelson.

- jlrun(code)

## ğŸ“¥ Input argument

- code - a string scalar, string array, character vector, character array.
- jlName, jlValue - Input arguments name and value
- outputs - string array: Julia variable names.

## ğŸ“¤ Output argument

- outvars - One or more Nelson workspace variable names returned as valid Julia types.

## ğŸ“„ Description

<b>jlrun(code)</b> function executes Julia statements present in the code string within the Julia interpreter.

Variables generated by the<b>jlrun</b> function remain persistent, allowing their usage in subsequent<b>jlrun</b> calls.

<b>outvars = jlrun(code, outputs)</b> Julia variables specified in outputs are returned to Nelson.

The values of these variables are captured in<b>outvars</b>.

<b>outvars = jlrun(code, outputs, jlName, jlValue)</b>, the<b>code</b> is executed with assigned input and output variable names using Nelson data passed through one or more name-value arguments.

