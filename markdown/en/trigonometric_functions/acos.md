# acos

Computes the inverse cosine in radians for each element of x.

## Syntax

- res = acos(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

acos computes the inverse cosine in radians for each element of x.

## Example

```matlab
A = eye(3, 3);
res = acos(A)

```
