# acsch

Inverse hyperbolic cosecant.

## Syntax

- res = acsch(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

<b>acsch</b> computes the inverse hyperbolic cosecant for each element of <b>x</b>.

## Example

```matlab
X = [3*pi, 2*pi, pi, 0];
R = acsch(X)
```
