# sec

Secant of angle in radians.

## Syntax

- res = sec(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

<b>sec</b>computes the secant of argument in radians for each element of<b>x</b>.

## Example

```matlab
x = -pi:0.75:pi;
R = sec(x)
```
