# acoth

Inverse hyperbolic cotangent.

## Syntax

- res = acoth(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

        acoth computes the inverse hyperbolic cotangent.

## Example

```matlab
A =  [1+2i, 2, -3];
res = acoth(A)
```
