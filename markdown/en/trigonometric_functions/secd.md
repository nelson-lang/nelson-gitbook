# secd

Secant of argument in degrees.

## Syntax

- res = secd(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

<b>secd</b>computes the secant of argument in degrees for each element of<b>x</b>.

## Example

```matlab
R = secd([1, 10+3i, 15+2i, 35+i])
```
