# asecd

Inverse secant of argument in degrees.

## Syntax

- res = asecd(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

<b>asecd</b> computes the inverse secant of argument in degrees for each element of <b>x</b>.

## Example

```matlab
R = asecd([1, 10+3i, 15+2i, 35+i])
```
