# acsc

Inverse cosecant in radians.

## Syntax

- res = acsc(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

        acsc computes the inverse cosecant of argument in radians for each element of x.

## Example

```matlab
R = acsc(3)
R = acsc(0.5)
```
