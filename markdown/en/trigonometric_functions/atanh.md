# atanh

Computes the inverse hyperbolic tangent.

## Syntax

- res = atanh(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

<b>acos</b> computes the inverse hyperbolic tangent.

## Example

```matlab
A =  [1+2i, 2, -3];
res = atanh(A)
```
