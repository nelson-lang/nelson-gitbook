# tanh

Computes the hyperbolic tangent in radians for each element of x.

## Syntax

- res = tanh(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

        tanhcomputes the hyperbolic tangent in radians for each element of x.

## Example

```matlab
A = eye(3, 3);
res = tanh(A)
```
