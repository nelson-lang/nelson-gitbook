# cot

Cotangent of angle in radians

## Syntax

- res = cot(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

<b>acot</b>computes the cotangent of angle for each element of<b>x</b>.

## Example

```matlab
R = cot([-i pi+i*pi/2 -1+i*4])
```
