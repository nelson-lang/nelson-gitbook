# coth

Hyperbolic cotangent.

## Syntax

- res = coth(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

        csch computes the hyperbolic cotangent for each element of x.

## Example

```matlab
X = [3*pi, 2*pi, pi, 0];
R = coth(X)
```
