# cos

Computes the cosine in radians for each element of x.

## Syntax

- res = cos(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

<b>cos</b>computes the cosine in radians for each element of<b>x</b>.

## Example

```matlab
A = eye(3, 3);
res = cos(A)
```
