# sech

Hyperbolic secant.

## Syntax

- res = sech(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

        sech computes the Hyperbolic secant for each element of x.

## Example

```matlab
X = [3*pi, 2*pi, pi, 0];
R = sech(X)
```
