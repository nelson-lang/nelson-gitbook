# acosh

Inverse hyperbolic cosine.

## Syntax

- res = acosh(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

<b>acos</b> computes the inverse hyperbolic cosine.

## Example

```matlab
A =  [1+2i, 2, -3];
res = acosh(A)
```
