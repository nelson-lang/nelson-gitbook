# acosd

Inverse cosine in degrees.

## Syntax

- res = acosd(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

        atand computes the inverse cosine in degrees for each element of x.

## Example

```matlab
x = [1 -20 0 2 5];
y = acosd(x)
```
