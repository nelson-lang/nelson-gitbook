# cosh

Computes the hyperbolic cosine in radians for each element of x.

## Syntax

- res = cosh(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

        cosh computes the hyperbolic cosine in radians for each element of x.

## Example

```matlab
A = eye(3, 3);
res = cosh(A)
```
