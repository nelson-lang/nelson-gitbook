# tan

Computes the tangent in radians for each element of x.

## Syntax

- res = tan(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

<b>tan</b>computes the tangent in radians for each element of<b>x</b>.

## Example

```matlab
A = eye(3, 3);
res = tan(A)
```
