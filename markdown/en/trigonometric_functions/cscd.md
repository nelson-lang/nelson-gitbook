# cscd

Cosecant of argument in degrees.

## Syntax

- res = cscd(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

        cscd computes the cosecant of argument in degrees for each element of x.

## Example

```matlab
R = cscd([35+i 15+2i 10+3i])
```
