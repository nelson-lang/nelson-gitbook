# sin

Computes the sine in radians for each element of x.

## Syntax

- res = sin(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

<b>sin</b> computes the sine in radians for each element of <b>x</b>.

## Example

```matlab
A = eye(3, 3);
res = sin(A)
```
