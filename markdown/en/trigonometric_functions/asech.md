# asech

Inverse hyperbolic secant of angle in radians.

## Syntax

- res = asech(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

        asech computes the inverse hyperbolic secant of argument in radians for each element of x.

## Example

```matlab
x = -pi:0.75:pi;
R = asech(x)
```
