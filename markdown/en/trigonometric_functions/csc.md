# csc

Cosecant of input angle in radians.

## Syntax

- res = csc(x)

## Input argument

- x - a numeric value

## Output argument

- res - a numeric value

## Description

        csc computes the cosecant of argument in radians for each element of x.

## Example

```matlab
R = csc(-pi+0.01:0.01:-0.01)
```
