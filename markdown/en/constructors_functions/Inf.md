# Inf

Infinity

## Syntax

- Inf
- inf
- Inf(n)
- Inf(n, m)

## Input argument

- n - a variable: n-by-n matrix
- m - a variable: n-by-m matrix

## Description

  <p><b>NaN</b> returns the IEEE symbol Inf (Infinity).</p>

## Examples

```matlab
Inf
```

```matlab
-Inf + Inf
```

```matlab
1.e1000
```
