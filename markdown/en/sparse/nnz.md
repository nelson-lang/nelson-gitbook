# nnz

Return the number of nonzero elements.

## Syntax

- v = nnz(M)

## Input argument

- M - a matrix: double or logical, sparse or not.

## Output argument

- v - a integer value.

## Description

  <p><b>nnz</b> returns the number of non zero elements in an matrix.</p>

## Example

```matlab
I = [1 2 3];
J = [3 1 2];
V = [32 42 53];
sp = sparse(I, J, V, 5, 4, 10)
size(sp)
nnz(sp)
nzmax(sp)
```
