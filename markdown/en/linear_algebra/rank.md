# rank

Rank of matrix.

## Syntax

- r = rank(A)
- r = rank(A, tol)

## Input argument

- A - matrix: double or single
- tol - tolerance

## Output argument

- r - a numeric value: a scalar.

## Description

  <p><b>rank(A)</b> returns the number of linearly independent columns in a matrix (rank of the matrix).</p>

## Example

```matlab
X = rand(10, 10);
r = rank(X)
```
