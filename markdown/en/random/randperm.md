# randperm

Random permutation of integers values.

## Syntax

- p = randperm(n)

## Input argument

- n - Number of integers in sample interval (positive integer).

## Output argument

- p - a row vector.

## Description

  <p><b>p = randperm(n)</b> returns a row vector containing a random permutation of <b>1:n</b>.</p>

## Example

```matlab
randperm(7)
```
