# iskeyword

Returns all Nelson keywords.

## Syntax

- state = iskeyword(name)
- ce = iskeyword()

## Input argument

- name - a string.

## Output argument

- state - a logical: true if is an Nelson keyword.
- ce - a cell of strings: list of Nelson's keywords.

## Description

<p>
            iskeyword returns a list of all Nelson keywords.</p>

## Example

```matlab
iskeyword('for')
ce = iskeyword()
```
