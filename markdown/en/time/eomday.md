# eomday

Returns last day of month.

## Syntax

- E = eomday(Y, M)

## Input argument

- Y - year: real.
- M - month: real.

## Output argument

- E - last day of month: real.

## Description

<p>
            E = eomday(Y, M) returns the last day of the month M for the year Y.</p>

## Example

```matlab
eomday(1900, 1:12)
```
