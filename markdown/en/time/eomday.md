# eomday

Returns last day of month.

## Syntax

- E = eomday(Y, M)

## Input argument

- Y - year: real.
- M - month: real.

## Output argument

- E - last day of month: real.

## Description

  <p><b>E = eomday(Y, M)</b> returns the last day of the month <b>M</b> for the year <b>Y</b>.</p>

## Example

```matlab
eomday(1900, 1:12)
```
