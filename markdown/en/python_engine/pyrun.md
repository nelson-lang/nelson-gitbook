# pyrun

Run Python statements from Nelson.

## Syntax

- pyrun(code)
- outvars = pyrun(code, outputs)
- outvars = pyrun(code, outputs, pyName, pyValue)

## Input argument

- code - a string scalar, string array, character vector, character array or python code object.
- pyName, pyValue - Input arguments name and value
- outputs - string array: Python variable names.

## Output argument

- outvars - One or more Nelson workspace variable names returned as valid Python types.

## Description

<p>
            pyrun(code) function executes Python statements present in the code string within the Python interpreter.</p>

<p>Variables generated by the pyrun function remain persistent, allowing their usage in subsequent pyrun calls.</p>

<p>
                outvars = pyrun(code, outputs) Python variables specified in outputs are returned to Nelson.</p>

<p>The values of these variables are captured in outvars.</p>

<p>
                    outvars = pyrun(code, outputs, pyName, pyValue), the code is executed with assigned input and output variable names using Nelson data passed through one or more name-value arguments.</p>

## Examples

```matlab
pyrun('a = b * c', 'b', 5, 'c', 10)
r = pyrun('d = a + c', 'd')
```

```matlab
pyrun(["a = 3","print(a)"])
```

```matlab
[R1, R2] = pyrun("a=b*c",["a","b"], 'b', 5, 'c', 10)
```

Python code object representing a script generated through the built-in compile function in Python

```matlab
PYCODE = pyrun('X = compile(''Y = 3'', ''test'', ''exec'')', 'X')
y = pyrun(PYCODE, 'Y')
```
