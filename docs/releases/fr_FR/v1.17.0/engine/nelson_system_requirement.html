Exigences système
Exigences système par plateforme.
📄Description

Linux :

Système d'exploitation : Ubuntu 24.04 LTS, Ubuntu 22.04 LTS, Ubuntu 20.04 LTS, Fedora, ArchLinux, NixOs

Processeur : tout processeur Intel ou AMD x86-64.

Mémoire (RAM) : voir les exigences du système d'exploitation (16 Go recommandés ou plus).

Stockage : 1 Go pour l'installation de tous les produits ; un SSD/NVMe est recommandé.

Graphisme : un GPU avec accélération matérielle et support OpenGL et au moins 1 Go de mémoire est recommandé.

Windows :

Système d'exploitation : Windows 10, 11 ou Windows Server 2022.

Processeur : tout processeur Intel ou AMD x86-64 avec jeu d'instructions AVX2 (CPU publié à partir de 2015).

Mémoire (RAM) : voir les exigences du système d'exploitation (16 Go recommandés ou plus).

Stockage : 1 Go pour l'installation de tous les produits ; un SSD/NVMe est recommandé.

Graphisme : un GPU avec accélération matérielle et support OpenGL et au moins 1 Go de mémoire est recommandé.

MacOs :

Système d'exploitation : macOS Tahoe, macOS Sequoia, macOS Sonoma, macOS Ventura.

Processeur : tout processeur Intel supporté par Apple et tout processeur de la série M.

Stockage : 1 Go pour l'installation de tous les produits ; un SSD/NVMe est recommandé.

Graphisme : tout Mac capable d'exécuter macOS Ventura dispose d'un GPU pouvant exécuter Nelson.
