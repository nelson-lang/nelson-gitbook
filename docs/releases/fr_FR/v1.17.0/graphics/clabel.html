clabel
Étiquetage des contours
📝Syntaxe
clabel(C,h)
clabel(C,h,v)
clabel(C)
clabel(C,v)
tl = clabel(...)
clabel(...,Name,Value)
📥Arguments d'entrée
Paramètre Description
C

Matrice Contour retournée par contour, contourf, oucontour3. Si vous passez un objet contourh, vous pouvez passer [] pour C.
h

Handle d'objet contour retourné par contour / contourf / contour3. Lorsqu'il est fourni, l'étiquetage utilise les informations attachées à l'objet contour (niveaux et matrice de contour).
v

Vecteur des niveaux de contour à étiqueter. Lorsqu'il est fourni, seuls ces niveaux reçoivent des étiquettes.
📤Arguments de sortie
Paramètre Description
t

Objets Text créés par clabel. Les propriétés String contiennent les valeurs de contour affichées.
tl

Objets Text et ligne créés lorsque des marqueurs droits sont utilisés (pour l'utilisation de style clabel(C)).
📄Description

La fonctionclabel insère des étiquettes dans les graphiques de contours :

  • Fournir une matrice de contour C et un objet de contourh pour étiqueter le texte tourné le long des lignes de contour.
  • Fournir uniquementC pour ajouter des étiquettes droites et des marqueurs '+' aux emplacements de contour.
  • Passer un vecteur de niveauxv pour étiqueter uniquement des valeurs de contour spécifiques.
  • Utiliser des paires Name,Value pour contrôler l'apparence du texte (un sous-ensemble des propriétés Text, plus LabelSpacing).
💡Exemples
Étiqueter les niveaux de contour (de base).
figure();
[x,y,z] = peaks;
[C,h] = contour(x,y,z);
clabel(C,h)
Étiqueter des niveaux de contour spécifiques.
figure();
[x,y,z] = peaks;
[C,h] = contour(x,y,z);
v = [2,6];
clabel(C,h,v)
Définir les propriétés des étiquettes de contour avec des paires Name,Value.
figure();
[x,y,z] = peaks;
[C,h] = contour(x,y,z);
clabel(C,h,'FontSize',15,'Color','red')
Étiqueter en utilisant uniquement la matrice de contour (étiquettes droites).
figure();
[x,y,z] = peaks;
C = contour(x,y,z);
clabel(C)
