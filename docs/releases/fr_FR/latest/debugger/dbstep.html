dbstep
Exécuter la ligne exécutable suivante lors du débogage.
📝Syntaxe
dbstep
dbstep in
dbstep out
dbstep nlines
📥Arguments d'entrée
Paramètre Description
nlines Entier positif spécifiant le nombre de lignes exécutables à exécuter. L'exécution s'interrompt aux points d'arrêt rencontrés en cours de route.
📄Description

dbstep exécute la ligne exécutable suivante du fichier courant lors du débogage, en sautant les points d'arrêt dans les fonctions appelées par cette ligne. L'exécution s'interrompt

dbstep in entre dans toute fonction appelée sur la ligne courante, s'interrompant à la première ligne exécutable de la fonction appelée.

dbstep out termine l'exécution de la fonction courante et s'interrompt juste après le retour à l'appelant. L'exécution s'interrompt aux points d'arrêt rencontrés en cours de route.

dbstep nlines exécute le nombre spécifié de lignes, s'interrompant à tout point d'arrêt rencontré.

💡Exemples
Passer par-dessus une fonction appelée.
function n = myfile(x)
  n = myfunction(x-1);
end
function z = myfunction(y)
  z = 2/y;
end
dbstop in myfile
myfile(2);
dbstep
Entrer dans une fonction appelée.
dbstop in myfile
myfile(2);
dbstep in
Sortir de la fonction courante.
dbstep out
Exécuter plusieurs lignes en une seule commande.
dbstep 4
