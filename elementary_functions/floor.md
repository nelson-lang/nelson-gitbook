floor

Round down

## Syntax

- C = floor(A)

## Input argument

 - A - a variable

## Output argument

 - C - result of floor.

## Description


  <p><b>floor</b> returns an integer matrix made of nearest rounded down integers.</p>


## Example

```Nelson
floor(pi)
```
